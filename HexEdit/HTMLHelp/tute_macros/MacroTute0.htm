Keystroke Macro Tutorial

 

Welcome to the HexEdit Keystroke Macro Tutorial.

 

What are Keystroke Macros?

 

Keystroke macros allow you to automate repetitive tasks.  You "record" a series of keystrokes so that they can be later "played back" as many times as you like.

 

When combined with other powerful features such as the calculator you can quickly perform complex tasks that might take hours or days to write a specialised program to perform.  Moreover, you can save them to macro files so that you can use them again later or give them to colleagues who have the same need.

 

Who Should Use this Tutorial?

 

This tutorial is for anyone who wants to get more out of using HexEdit.  If you find you often have to perform simple (but tedious) repetitive tasks with HexEdit then using macros can save you a lot of time.

 

For more complex tasks it can be quicker and easier to create a HexEdit macro to do the job than to write a special program to do it.  Though sometimes it takes a bit of trial and error to get keystroke macros just right.

 

Many users ask us to add special features HexEdit, for a specific problem they have.  They are often surprised when they find that what they require can be done simply with a keystroke macro.

 

It should take less than 40 minutes to complete the tutorial.

 

Before You Start...

 

It is assumed that you are reasonably familiar with HexEdit and tools such as the calculator.  You do not need to know how to create keystroke macros.

 

What You will Learn

 

In this tutorial you will learn how to record and run a macro.  You will also learn how to save a macro to a disk file and run such a macro.  Some of the macro commands we use include creating a new file, performing operations using data from multiple files and displaying informational messages to the user.

 

There are also more advanced techniques like creating a loop using a nested macro.  And we will use the calculator to manipulate file data and move the cursor (current insert position) in the active file.

 

There are five steps:

 

1. Getting Started

2. Creating a macro to XOR bytes

3. Saving the macro to disk

4. Creating the main macro

5. Saving and testing

 

At the bottom of each step there is a "Next" button, which allows you to move to the next step (and a "Previous" button to allow you to go backwards).

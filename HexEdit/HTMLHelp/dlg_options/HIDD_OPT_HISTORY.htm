History Lists Dialog

 

 

This page allows you to control history lists used (for convenience) throughout HexEdit.  It also allows for clearing of other remembered information for privacy reasons. If other people have access to the same computer you may want to keep this information private.  Note that when files (bookmarks file, etc) are cleared they are first overwritten with zeroes before being deleted, so that they cannot be recovered using an "undelete" utility.

 

Never add files opened to "My Recent Documents List"

Whenever you open a file in a program Windows usually remembers the name of the file and adds it to its recently opened files list, which is available through the start menu as a sub-menu under "My Recent Documents".  This is a useful capability for most applications but may not be appropriate for files you open in HexEdit.

 

Recently used file list entries

HexEdit remembers all the files that you open.  This option says how many of the most recently opened files to show on the File menu.  You can use any value from 1 to 16.

 

Find list size

This is the size of the history list which stores recent searches.  This is used for the history list for the Find tool (see Toolbar Tools) and also for the history list in the Find dialog.

 

Replace list size

This is the size of the history list for replacements in search and replace operations (see Replace page).

 

Decimal jumps list size

This is the size of the history list used in the decimal jump tool (see Toolbar Tools).  The list is also available in the Calculator.

 

Hex jumps list size

This is the size of the history list used in the hex jump tool (see Toolbar Tools).  The list is also available in the Calculator.

 

Explorer filters list size

This is the size of the history list used for filters used to filter the files seen in the Explorer window.  This list contains both filters types in and those selected from the predefined filters list.

 

Explorer folders list size

This is the size of the history list used for folders visited in the Explorer window.  It contains the names of folders typed in and also those moved to by navigating around the folder hierarchy.

 

List sizes are the number of entries saved when you exit HexEdit, not the size of the list when HexEdit is running.  To clear the list set the size to zero.

 

Clear recent file list

This will completely clear the file containing the list of recently opened files.  The recent files on the File menu are also removed and all entries that appear in the Recently Opened Files dialog disappear.  Note that formatting information (character set, font, etc) and highlighting are also deleted when the recent file list is deleted as this information is stored in the same file.

 

Clear all bookmarks

Clears all bookmarks that appear in the Bookmarks dialog.

 

Clear selected lists on exit

Whenever you exit HexEdit normally all the above items that are checked are deleted.  If HexEdit is terminated abnormally (say with Task Manager) then the lists are not cleared.

 

Clear Selected Lists Now

Clears the selected lists immediately.

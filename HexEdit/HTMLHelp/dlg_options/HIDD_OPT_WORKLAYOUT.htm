Workspace Global Options Page

 

 

This page allows you to change some global options not specific to the document windows.

 

Restore main window position on startup

The main HexEdit Pro window is restored to its position when you last ran the program.  This includes a maximized window, but a minimized window will be restored to its prior position.

 

More Startup Options

Jumps to the System General page which has startup options (including Restore Main Window Position).

 

Display Window Tabs

When this options is selected, tabs are displayed at the top or bottom of the main HexEdit Pro window that show all the files and windows currently open.  Each tab shows an icon for the file type and the file name.  To switch to any window just click on its tab.

 

Window Tabs at Bottom

When this checkbox is on window tabs (see above) are displayed at the bottom of the main HexEdit Pro window.  Otherwise they are displayed at the top.

 

Show Icons in Tabs

Shows an icon for the file.  The icon is obtained from Windows and comes from the file or from the .EXE file for which the file type is associated.

 

Allow modeless dialogs to dock

The modeless dialogs (Calculator, Find dialog, Bookmarks, Properties and Explorer Window) can remain open while working in HexEdit Pro for quick access. Normally they float in front of the main HexEdit window, but can be allowed to "dock" with the sides of the main window so they do not obscure any other parts of the display.  If this option is on, when you move the dialog (by clicking and dragging on the title bar) over the left or right side of the main screen it will be docked where you drop it.

 

This option is initially off as some people can become confused when a window is dragged and docks unintentionally.  (However, you can prevent a floating window from being docked unintentionally by holding down the Ctrl key while dragging.)

 

Undocked dialogs move out of the way

Modeless dialogs when floating can obscure parts of the main window.  Normally, if you move the cursor in the active window under a modeless dialog then HexEdit Pro will move the dialog out of the way so you can see what you are doing.  Most people find this extremely convenient but one person requested that this could be turned off..

 

Hex in uppercase

When this option is selected the digits A to F in all hex numbers are displayed in uppercase. This affects all windows, dialogs, the Find Tool, the Hex Address Tool, etc. This option is recommended as there is less chance of confusing certain digits (such as "6” and lowercase “b”), depending somewhat on the font you are using.

 

Numeric abbreviations

HexEdit Pro creates abbreviations for large integers in many places -- for example, for files sizes in the Recent File List, Explorer windows, file sizes in Summary Properties page, etc. When measuring sizes of memory (or anything that comes in sizes of multiples of powers of 2) the letter K can be used as an abbreviation for 1024 (2^10) rather than the more common use of K (for kilo) meaning 1000.  Similarly M can mean 1000K or 1024K.  There is no common practice for these abbreviations (in fact even the programs that come with MSWindows are inconsistent) so HexEdit Pro allows you to select your preference.

 

[Personally I use the 2nd option (K=1024, M=1000K, G=1000M) as 1024 only differs from 1000 by 2.4% (and this is a common convention) but 1024*1024 differs from 1,000,000 by 4.8% and this is getting a bit too big.]

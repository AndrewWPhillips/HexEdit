Colour Options Page

 

 

This page allows you to change the colours used in the active window display and to create and save colour "schemes".  You can also display different groups of hex values in different colours, which for many types of files can be useful for easily spotting patterns and irregularities.

 

The left side of the page displays the current schemes with buttons to create new schemes, and delete or reset the currently selected scheme.  The top four schemes are standard schemes that cannot be deleted.

 

The four standard schemes are used automatically to display different characters sets.  So if the current character set is ASCII (see Char Set ASCII) and the current scheme is "ASCII default" then changing to the EBCDIC character set (see Char Set EBCDIC) automatically changes to the "EBCDIC default" colour scheme.

 

 

The right side of the page displays the available colours for the currently selected scheme.  The top six colours are standard ones that cannot be deleted, or reordered.  Below the standard colours are one or more entries that specify colour groups that can be created, deleted and reordered.

 

To use different colours you need to split the range of byte values (0-255) into groups. A group can consist of individual values or discontiguous ranges. For example, to use the values found in normal ASCII text files use “9,10,13,32-126”, which are the values of tab, line-feed, carriage-return and the range of normal display characters. As well as the values each group has an associated name and, of course, a colour.

 

When displaying the hex byte HexEdit scans the groups from the top and uses the colour of the first group the value is found in. If a value appears in more than one group the colour from the higher group is used. You can use the arrow buttons to change the order of the groups. Also if a value is not assigned to any group then that value is not displayed. (Actually it’s just displayed using the background colour.)

 

The new scheme button invokes the New Scheme Dialog which allows you to create a new scheme based on any existing scheme.

To delete a scheme select it in the list below and click this button.  You cannot delete the first 4 standard schemes but you can, of course, change their colours.

Reset

This button changes all the colours of the currently selected scheme back to the default.  The default colours for the first four (standard) schemes are all slightly different.  Reseting any other scheme sets it to be the same as the default "ASCII default" scheme.

List of Schemes

Select a scheme from the list before you delete, reset or modify it.

 

Selecting a scheme and clicking OK selects that scheme for use in the active window, if any.

 

Name

The description of the currently selected group. You can also type the name of a new group to be added here.

List of Names

The top 10 entries are for standard colours that every colour scheme must have.  Below that are the names of all the current groups. To change the colour of any of the entries first select it in this list before using the colour selection tool (below).  Note that version 3.0 adds separate colors for the mark and bookmarks background.

 

Background

The background colour of the window, usually white or a very light shade.

Mark

Background colour for the byte at the "mark".  See Set Mark Command.

Highlighting

Background colour of bytes that have been highlighted.  See Highlighter.

Bookmarks

Background colour of bytes that have been bookmarked.  See Edit Bookmarks.

Search occurrences

Background colour of all occurrences of the current search text (when background searches are enabled).  (See Background search option in System Options page.)  Also used for search occurrence progress bar in the status bar.

Change tracking

Used to show replacements, insertions and deletions.  See Track Changes command.

Address background

Background colour of the address area.

Sector boundary

Used to show the boundary of disk sectors and background for bad sectors if the "Sector Bounds" option is on in the Window Display Options page.

Hex addresses

Addresses are shown in this colour if the Decimal addresses toggle is off.  Also used for hex addresses in the status bar and hex jump tool.

Decimal addresses

Addresses are shown in this colour if the Decimal addresses toggle is on.  Also used for decimal addresses in the status bar and decimal jump tool.

Standard Colours and Uses
 

Below the Decimal Addresses colour are one or more colours used to display the bytes of the file (both hex and characters).  Select one of these before adjusting its colour or range of values using the controls below.

Colour

Select a colour from the drop down list to change the colour of the currently selected entry in the list above.  You can select one of the 20 colours or "Automatic" to choose a colour consistent with your "Windows Display Appearance Scheme".  Select "Other" to see a dialog with two pages: the "Standard" page allows you to easily select from 140 standard colours; while the "Custom" page allows you to enter any colour based on RGB or HSV values.

Values

Enter the values for the current or new group. Values are in decimal and comma-separated; a range of values may be entered by typing two values separated by a dash or a colon.

The new colour group button adds a group to the list with the name currently in the Name field (see above), and the currently selected colour and range of values. You cannot Add a group with the same name as an existing group.

This removes the current group from the list. Before removing you must select the group to remove from the list.  You cannot remove any of the first 10 standard items from the list.

 

If you accidentally remove a group, simply click the new colour group button before you do anything else. This reinserts the same group. You may then need to use the up arrow button to reposition the group where it was before.

The order of the groups is significant. The up arrow allows you to move the current group up in the list.  You cannot move any of the first 10 standard items.

The down arrow allows you to move the current group down in the list.

File Type Filters Page

 

 

When you use HexEdit Pro to select a file you are presented with one of the standard Windows file dialogs.  The File Open dialog is used in the File Open command, the Read File Command and the Append Selection to File.  The File Save As dialog is used in the File Save As command, and the Write Selection to File command.  These dialogs contain a drop down list ("Files of Type") that allows you to select the type of files you see using filters operating on the file names.

 

In most programs this filter list is fixed (ie, can't be modified) as applications typically operate on only one or a few different types of files.  As HexEdit Pro can be used with any type of file, it is very useful to be able to control the list of filters that are available.  This is what the Filters Page does.

 

List of Filters

The "Filter name" column is the name of the filter that is displayed in the drop-down list.  The "Filter" column determines which files are displayed -- for example "*.exe;*.dll" will restrict the file dialog to only show files with file name extension of .EXE or .DLL.  The left-most column contains check boxes that determine whether the filter is in use -- this allows you to turn off a filter without actually deleting it from the list.

 

Adds a new filter at the current place in the list.  It is enabled by default but you need to enter a name and filter specification.

 

Deletes the currently selected filter in the list.  It is recommended that you just disable and not delete filters in case you need them again later.

 

Moves the filter up in the list unless it is already at the top.  The order in this list determines the order the filters are displayed in the "Files of Type:" list that appears at the bottom of the standard Windows file open and save dialogs.

 

Moves the filter down in the list unless it is already at the bottom.  It is recommended that the "All Files" filter is kept active and at the top of the list.

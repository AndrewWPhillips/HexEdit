Printer Decorations Page

 

 

The Printer Options page gives you a great deal of control over how pages are printed.  Not only can you set margins and line-spacing but you can created custom headers and footers with substitutable parameters (see below).

 

Header

The header is printed at the top of each page.  It is divided into a left section, a centred section and a right section.  Separate each section by a vertical bar (|).  If you do not supply a vertical bar the header is all left justified.

 

The header can contain any text and/or any of the substitutable parameters (see the table below) which you can easily add using the arrow button.  The header is displayed within the top margin so make sure enough room is provided.  You can position the header with the Header (From Edge) control mentioned above.

 

First page

When this checkbox is selected you can specify a different header on the first printed page.  To remove the header altogether from the first page just leave the text blank.  The options are exactly the same as for Header (above).

 

Footer

The footer is similar to the header but is printed at the bottom of each page.  It is divided into a left section, a centred section and a right section.  Separate each section by a vertical bar (|).  If you do not supply a vertical bar all of the footer is left justified.

 

The footer can contain any text and/or any of the substitutable parameters (see the table below) which you can easily add using the arrow button.  The footer is printed within the bottom margin so make sure enough room is provided.  You can position it with the Footer (From Edge) control mentioned above.

 

First page

When this checkbox is selected you can specify a different footer on the first printed page.  To remove the footer altogether from the first page just leave the text blank.  The options are exactly the same as for Footer (above).

 

Swap left/right sections for even pages

The header and footer strings above are each separated into three sections separated by two vertical bars like this: "LEFT | MIDDLE | RIGHT".  Normally the text in the sections are printed on the left side, center, and right side of the page.  When this option is selected every second page (i.e. all even page numbers) have the left and right sections reversed.

 

This settings applies to all the above header and footer options.  It is typically used for duplex (double-sided) printing.

 

Watermark

Select this option to have large light gray text printed diagonally on every page, "behind" all the other text.  The same parameters as for the header and footer are permitted (see table below), or you can just used some fixed text like "CONFIDENTIAL".

 

 

Param

When printed is replaced with:

&F

The file name

&A

The full path name of the file

&P

The current page number

&D

The current date

&T

The current time

&N

The current date and time in long format

&C

The date and time the file was created

&M

The date and time the file was last modified

&G

Anything you have entered into the category field for the file

&K

Anything you have entered into the keywords field for the file

&C

Anything you have entered into the comments field for the file

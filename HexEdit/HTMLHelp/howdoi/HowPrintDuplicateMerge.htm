How Do I Stop Printing Lots of Null Bytes?

 

Sometimes you need to print a large section of a file or all of a file but most of the file is full of null bytes (or other repeated bytes such as spaces).  Instead of wasting lots of paper HexEdit provides an option where duplicate lines are not printed multiple times but the line is printed once and a count of the number of following repetitions is printed underneath.

 

To do this select the bytes you want to print (or press Ctrl+A to select the whole file) then in the Print Dialog use the option to print the selection in the "Print Range" box.  Then select the "Merge and Count repeated lines" check box.

 

This option is also useful to prevent printing repeated records in a fixed length record file, as long as the line length is set to the length of the records.

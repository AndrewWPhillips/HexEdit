How Do I Make a Large Selection in a Huge File?

 

HexEdit allows you to edit files of more than 4 Gbytes if you have an NTFS filesystem (under NT/2000/XP). For files this large the scrollbar resolution makes the vertical scrollbar virtually useless for moving to a particular address.  For huge files the method for selecting a large block of bytes as explained in How Do I Make a Large Selection? becomes too cumbersome if not impossible.  In this case you need to use the following method.  Note that the instructions below assume the default keyboard setup.

 

Double-click on the first byte of the selection to set the mark (see the Set Mark Command).  Then use the hex or decimal address tool to jump to the end of the desired selection.  For example, press Alt-J type ABCD and press Enter to jump to address ABCD (hex).  Then use the Extend To Mark Command (menu item: Edit/Mark/Extend to mark) to select all the bytes from the mark to the cursor.

 

You may then perform operations on the selection such as writing it to a file.  Of course, such operations may be slow to complete if you have a very large selection.

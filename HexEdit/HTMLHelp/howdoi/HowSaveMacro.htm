How Do I Save a Macro?

 

Complex keystroke macros are difficult to get right.  So once you have a macro working exactly right you may want to save it so you can use it again.

 

To save the current keystroke macro open the Macro page of the Options Dialog (use the Tools/Options menu item) and click the "Save Current Macro" button.  This displays the Save Macro dialog where you can then enter a file name and various options stored with the macro.  You can also enter a comment that is displayed when the macro is later reloaded - this can remind you of what the macro does and any requirements it has (such as initial conditions).

 

HexEdit macro files are saved with the extension .HEM in the current macro folder.  You can change the macro folder using the "Change Macro Folder" button in the Macro Page.

 

You can later load a macro from file again using the "Load Macro" button in the Macro Page.  If you save to the special file name "AUTOEXEC.HEM" the macro is loaded and run each time you run HexEdit.  If you save with a name of a file-extension preceded by an underscore then the macro will be run whenever a file of that type is opened -- for example _EXE.HEM will be run whenever a .EXE file is opened.

How Do I Edit Toolbar Buttons?

 

You can edit the picture on any toolbar button.  To do this you must have the Customize dialog open (use the Tools/Customize menu item).  Right-click the toolbar button and select the "Button Appearance" item from the popup menu to bring up the Button Appearance dialog.

 

You can also create a toolbar button for a command that does not already have one.  For example, many menu commands do not have images.  Also non-menu commands that are, by default, just available through keyboard shortcuts can be added to a toolbar.  To do this drag the command from the Commands page of the Customize dialog onto a toolbar and select the "Button Appearance" item from the popup menu as above.

 

In the Button Appearance dialog, select to display the button as "Image Only" or "Image and Text" then select "User-defined Image".  You can then create you own image using the "New" button, or select from one of the predefined buttons or any buttons you have previously drawn.  You can use one of the pre-defined buttons as a starting point by selecting it and clicking the "Edit" button.

 

The button editor (invoked by the "New" or "Edit" button) provides basic editing tools such a freehand and line-drawing, color-selection, fill, circle/ellipse, eye-dropper, clear etc.  You can also create an image in another editor and paste it in using the paste button.

 

Note that any images you create or edit are kept in a file called ToolbarImages.bmp which is stored in the same directory as any saved keystroke macros.  If more than one person use the same computer and edit their toolbar buttons then they should use different folders.  See Macro Options page for how to change this location.

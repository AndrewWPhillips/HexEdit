How Do I Search for a Number?

 

To search for a number you need to know in what format it is stored.  That is how many bits (8, 16, 32 or 64), byte-order (big- or little-endian), and format (unsigned, 2's complement signed etc).  If you are not sure then remember that number could also be stored as a floating point value (see How Do I Search for a Floating Point Number?) or even as ASCII characters (in which case just do an ASCII search with the Find command).

 

If the file containing the number was created on a Windows 95/98/NT etc then it will probably be a little-endian 32 bit value.  But very small numbers may be stored in single byte.  Files created by MSDOS/16 bit Windows often use 16 bit numbers, but may use 32 bit numbers for large values and 8 bit numbers for small values.  See Numbers for more information about number formats.

 

Once you know what format the number is in you can search for it using the Number page of the Find dialog.  If you don't know the format you can try different ones as well as searching for the number as text.

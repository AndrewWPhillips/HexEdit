How Do I Abort a Search?

 

Searches in HexEdit are very fast but if you are searching a file of many Megabytes or Gigabytes or the file is located on a slow network it can take a long time.  Imagine you are looking for something near the start of a 60 Gigabyte file but after you typed the string and started the search you realise you typed the wrong thing and it will never be found.  Instead of waiting minutes or hours for the search to complete you need to cancel the search.

 

You can stop a search at any time by pressing the Escape key.  You will be asked if you want to continue in case you pressed the Escape key by mistake.  You can also use the Escape key to terminate a comparison or a keystroke macro.

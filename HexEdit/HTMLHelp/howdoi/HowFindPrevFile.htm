How Do I Find a File That I Opened Before?

 

You can select a previously opened file from the recently opened file list in the file menu.  The list is kept in MRU (most recently used) order.  If the file has fallen off the end of this list then you can check the list of all files that you have opened in the Recently Opened Files dialog.

 

The list of files in this dialog also defaults to MRU order.  If you can remember roughly when you last opened the file (in HexEdit) you can check the dates in the "Last Used" column.

 

If the list is long and you opened the file a long time ago it may be hard to find it.  This is where the ability to sort the list on the other columns is useful.  If you can remember the file name, the location, the size, or the type then sort on the "File Name", "Location", "Size" or "Type" column, by clicking on the column heading.  Click the heading again to reverse the order.

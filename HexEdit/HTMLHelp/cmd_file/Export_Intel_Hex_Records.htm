Export Intel Hex command

 

This command writes the current selection to an Intel Hex file.  As the addresses in these files are only 16 bits this command is disabled if the address of the last byte is greater than 65535 (which may depend on how the start address is calculated -- see below).  It is also disabled if there is no selection.

 

This command brings up the standard Save As dialog which allows you to save the file using any valid file name. If the file already exists you are prompted to overwrite it.  A Intel Hex file is a text file.

 

Each Intel Hex record contains an address and a certain number of bytes stored as text along with a checksum.  HexEdit will only create records that are contiguous, and will generate correct checksums.  The number of bytes per record and how the starting address is generated are determined by the Export section of the System Options page of the Options dialog.

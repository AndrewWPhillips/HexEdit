New command (File menu)

 

Use this command to create a new file.  (You can open an existing file with the Open command.)  It automatically invokes the New File Dialog, allowing you to create a file of any size and filled with repeating byte values, strings, numbers or random bytes.

 

The file created is given a generic default name, but when you attempt to save it with the File Save command the Save As dialog is automatically brought up.

 

If your default options (see "Save As Default" button in Window Display Options page) are set such that when you open a file it is opened in read-only mode then you will not be able to change the file until you turn off read-only mode (most easily done by double-clicking the "RO" pane in the status bar).

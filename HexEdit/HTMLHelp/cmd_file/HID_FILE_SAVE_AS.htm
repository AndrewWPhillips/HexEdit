Save As command (File menu)

 

Use this command to save the currently active file under a different name or in a different folder. You might do this if you want to modify a file but retain the original version. HexEdit displays the Save As dialog box so you can save the file under a different name.

 

To save a file with its existing name and folder, use the Save command. The information and warnings about backup files and large files mentioned under the Save command also apply here.

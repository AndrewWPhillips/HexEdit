Export Hex Text command

 

This command writes the current selection to a text file.  It writes each byte as a pair of hex digits, each pair separated by a space.  The number of pairs per line is determined by the Bytes per line option in the Export section of the System Options page of the Options dialog.  The command is disabled if there is no selection.

 

This command brings up the standard Save As dialog which allows you to save the file using any valid file name. If the file already exists you are prompted to overwrite it.

 

Addresses and checksums are not written as in Export Intel Hex Records and the Motorola S record exports.  If there is any chance of data corruption it is preferred to use the Export Motorola S3 Records which provides a checksum of each line, a record count record at the end of file to make sure that no complete lines were lost and (if importing into HexEdit again) a check that addresses are contiguous.

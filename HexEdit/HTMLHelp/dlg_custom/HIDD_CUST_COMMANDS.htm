Customize Commands Page

 

 

This page allows you to customize toolbars and menus. Click a category then drag an item from the “Commands” list to a toolbar or menu.

 

You can customize existing toolbars or create new toolbars. Use the “new” button on the Toolbars Page to create new toolbars.

 

You can customize the main menu or any of several context menus using the Menu page. Select the desired menu on the Menu Page then return to the Commands page and drag items from the Commands list to the menu.

 

To remove buttons, sub-menus etc from toolbars or menus click and drag them to an unused part of the screen, or right-click on the item (while the Customize dialog is open) and select “”Delete” from the context menu.

 

Categories

Selecting a category from the list displays all the commands for that category in the Commands list (below). The top 7 categories correspond to commands on the main HexEdit menus.

 

The “New Menu” category allows you to create a new menu or sub-menu on a toolbar or the Menu Bar. You can change the name of a new menu by right-clicking and selecting “Button Appearance” from the popup menu.

 

Some commands and tools do not appear on any menus but most can be found on the built-in toolbars. These are split into 3 categories:

  • Format commands: include commands for changing the format of the display and include tools for selecting the colour scheme, font and font size

  • Navigation commands: include jump and bookmark commands and include the jump hex, jump decimal and bookmark tools

  • Find commands: include search commands and options and the find tool

The “All Commands” category lists all commands in the “Commands” list.

Commands

Shows the buttons associated with the current category. Drag the item you want from the box to a toolbar or the menu bar.

Description

Provides a description of the selected command.

Customize Tools Page

 

 

This page allows you to add items to the Tools menu that when selected cause an external program to be executed. You can add up to nine tools to the list. Typing an ampersand (&) in the tool’s name makes the following letter a menu accelerator key.

 

By default this menu includes two entries that run the Windows Calculator (calc.exe) and NotePad (notepad.exe).

Menu Contents

Show the currently defined tools. Use the four buttons above this list to add new entries, delete entries or move entries up and down in the list. The fields below apply to the currently selected entry in this list.

Command

This is the full path name of the executable .to run when the Tools menu item is selected. Click the button to the right to browse and select the executable file to be run.

Arguments

These are the command line arguments to pass to the program. Use the argument “$(FilePath)” (without the quotes) to get the name of the file in the window that is active when the menu item is selected.

Initial directory

This is the directory in which the program is run.

Customize Toolbars Page

 

 

This page allows you to manage your toolbars and create custom toolbars. You can move toolbar (and menu bar) items within a toolbar or between different toolbars by clicking and dragging. You can add new items to these toolbars using the Commands Page.

 

The buttons on toolbars can also be customized by right-clicking them and selecting from the context menu. You can display the button as an image and/or as text. To change the image or text associated with the button select “Button Appearance” from the context menu.

 

You can even create your own button image using the built-in image editor using the New or Edit button. Note that all user-defined button images are store in a file called ToolbarImages.bmp. This is stored in the macros folder which by default is the same directory as HexEdit.exe is stored. If more than one user uses HexEdit on the same computer then the macros folder for these users should be set to different directories to avoid making conflicting changes to toolbar button images (see Macro Options).

 

Toolbar controls that include edit controls/drop lists (eg. the find tool and the hex and decimal address tools) can be resized. To do this, click the arrow on the right end of the tool, while the Customize dialog is open. You should then be able to resize the control by dragging it right side.

 

The Options Page also has facilities to change the behaviour of all toolbars.

 

Toolbars

This is the list of all toolbars. It includes the Menu Bar and the two built-in toolbars (Tool Bar and Edit Bar). The buttons to the right of the list operate on the currently selected toolbar. You can also display or hide any toolbar using the tick box next to its name. The Menu Bar cannot be hidden.

Reset

Can be used to reset a built-in toolbar (the Tool Bar or Edit Bar) or the Menu Bar. This button is disabled for custom toolbars.

Reset All

Resets all standard toolbars and the Menu Bar to their default state. This does not affect custom toolbars.

New

Creates a custom toolbar.

Rename

Allows you to rename the selected toolbar. You cannot rename built-in toolbars.

Delete

Allows you to delete the selected toolbar. You cannot delete built-in toolbars.

Show Text Labels

Forces text labels to be displayed for all buttons on the currently selected toolbar. This option is disabled for the Menu Bar as menus always have text labels. To change the text label for a button right-click the button and select “Button Appearance” from the context menu and change the “Button Text” field.

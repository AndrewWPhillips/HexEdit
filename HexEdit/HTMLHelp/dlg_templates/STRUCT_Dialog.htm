STRUCT Dialog

 

 

This dialog allows you to set the options of a template STRUCT element (and its contained elements, using the Edit button).  It is usually invoked by right-clicking the STRUCT node in the template tree view and selecting the "Edit..." item.

 

The STRUCT item acts as a container for one or more sub-elements of different types.

 

Name

This name identifies this specific field in the file.  The name is used in expressions to access sub-elements of the STRUCT (using the dot operator).  The name is required  (unless the parent is a FOR) so that you can access this element, and should be different to the names of siblings (if the parent is also a STRUCT).

 

If you need to show that different parts of the template have exactly the same data structure or type, you can give them both the same "Type name" (see below).

 

Elements

This is the list of elements currently contained in the STRUCT.  Select an element in the list before using the buttons next to the list (Edit etc - see below).  Select the blank entry at the bottom of the list before inserting in order to append to the list.

 

Edit...

This edits the sub-element of the STRUCT that is currently selected in the Elements list.  A "child" dialog appears near the dialog - you can't do anything in the STRUCT dialog until the child dialog is closed again.

 

Insert

Inserts a new element in the STRUCT just before the currently selected element in the Elements list.  To add an element to the end, select the blank line in the Elements list past the last one before clicking the Insert button.

 

Deletes the element currently selected in the Elements list.  Note that all sub-elements are also deleted, so accidentally deleting an element could lose a lot of work - for this reason you are prompted that you really want to delete it.

 

Moves the current element up in the Element list.

 

Moves the current element up in the Element list.

 

Elt size

This just displays the size of the currently selected element in the Elements list.  If the element (or any of its sub-elements) are variable in size then a question mark (?) is displayed instead.

 

Offset

Displays the offset within the STRUCT of the currently selected element in the Elements list.  If any elements above it are of variable size a question mark (?) is displayed.  To get the size of the entire STRUCT select the blank line at the end of the Elements list.

 

Type name

Generic type name for the data element.  This may be the name of a C/C++ struct/class if the file is manipulated by a C/C++ program.  This is a comment field only and has no effect on the behaviour of the template.

 

Display

This can be used to show the value of important field(s) in the tree column of the tree view.  This can be blank or a simple string but usually contains expressions enclosed in braces, with optional format specification.  See the section called Displaying Information in the Tree Column in Editing templates for more information.

<< and >> buttons

If the parent element is a STRUCT then these buttons are displayed in the bottom right of the dialog.  They allow you to move to the previous and next sibling (ie the previous and next elements in the parent STRUCT).  This is very handy when you quickly want to view or change values in all the children.

 

Using the button is equivalent to clicking OK then editing the previous or next element in the parent STRUCT.

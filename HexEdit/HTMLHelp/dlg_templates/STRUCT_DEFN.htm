Structure Definition Dialog

 

 

This dialog allows you to define a structure that can be used throughout the template.  It is similar to STRUCT Dialog, except that no structure is added to the template until an instance is created with Use Struct Dialog.  The strcuture definition can contain any element that a normal STRUCT can contain including using structure definitions -- recursive data structures can be created by using an instance of a structure definition within itself.

 

Structure definitions can only be added at the start of the top-level tree node.  To add one right-click the top level node and select the "Add Struct Definition" item of the popup menu.  To edit a structure definition just right-click it in the tree view and select the "Edit..." item.

 

Type name

This name identifies the structure definition so you can later use it.

 

Elements

This is the list of elements currently contained in the structure.  Select an element in the list before using the buttons next to the list (Edit etc - see below).

 

Edit...

This edits the sub-element of the structure that is currently selected in the Elements list.  A "child" dialog appears near the dialog - you can't do anything in the STRUCT dialog until the child dialog is closed again.

 

Insert

Inserts a new element in the STRUCT just before the currently selected element in the Elements list.  To add an element to the end, select the blank line in the Elements list past the last one before clicking the Insert button.

 

Deletes the element currently selected in the Elements list.  Note that all sub-elements are also deleted, so accidentally deleting an element could lose a lot of work - for this reason you are prompted that you really want to delete it.

 

Moves the current element up in the Element list.

 

Moves the current element up in the Element list.

 

Elt size

This just displays the size of the currently selected element in the Elements list.  If the element (or any of its sub-elements) are variable in size then a question mark (?) is displayed instead.

 

Offset

Displays the offset within the STRUCT of the currently selected element in the Elements list.  If any elements above it are of variable size a question mark (?) is displayed.  To get the size of the entire STRUCT use the offset of the blank line at the end of the Elements list.

 

Display

This can be used to show the value of important field(s) in the tree column of the tree view.  This can be blank or a simple string but usually contains expressions enclosed in braces, with optional format specification.  See the section called Displaying Information in the Tree Column in Editing templates for more information.

 

When a structure definition is used (see Use Struct Dialog) this value may be overridden.

IF Dialog

 

 

This dialog allows you to set the options of a template IF element (and its contained element, using the Edit button).  It is usually invoked by right-clicking the IF node in the template tree view and selecting the "Edit..." item.

 

An IF element is used for an optional element, or to select between two alternative elements.

 

Note that (unlike STRUCT, FOR etc) IF elements have no name.  For expression evaluation, the name of a sub-element of the IF can be used in expressions.  If during the evaluation of the template the IF expression evaluates true then the name of the first sub-element is used, otherwise if it evaluates false the name of the second (else part) sub-element is used, if present.  Typically, both sub-elements would have the same name (but not necessarily evaluate to the same result).

 

Condition

A boolean expression that says whether the contained sub-element is present.  You can use any value from the template so far processed.  Use the button with the triangle to see a menu (and possibly submenus) of all available values.

 

Note that if the expression evaluates to false, the sub-element is still shown in the tree (grayed out) in edit mode.  This allows you to edit the sub-element.

 

Element

This displays the name and/or type of sub-element contained in the IF.

 

Edit

Use this button to edit the element that is contained in the IF.  You cannot insert or delete elements as in a STRUCT since the IF must always have exactly one sub-element (apart from the ELSE part - see below).  However, you can replace it with an element of a different type (see below).

 

Replace with

Use this button to replace the element contained in the IF with an element of a different type.

 

Else

Use this check box to enable the else part.

 

Element

This displays the name and/or type of sub-element contained in the else part.

 

Edit

Use this button to edit the element that is contained in the else part.

 

Insert/Replace with

Use this button to add (if one is yet to be added) or replace the element contained in the else part.

<< and >> buttons

If the parent element is a STRUCT then these buttons are displayed in the bottom right of the dialog.  They allow you to move to the previous and next sibling (ie the previous and next elements in the parent STRUCT).  This is very handy when you quickly want to view or change values in all the children.

 

Using the button is equivalent to clicking OK then editing the previous or next element in the parent STRUCT.

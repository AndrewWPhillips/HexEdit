Choose Font dialog box

 

 

This dialog allows you to change the font, its style, size etc for the active window.  It is invoked with the Font command.

 

Font

Type the name of a font or select a font from the list of available fonts. You can choose any font but for HexEdit a non-proportional or monospaced font (where all characters are the same width) works best.  Unfortunately most fonts are proportional fonts, but Courier, Courier New, Fixedsys, Terminal, and any console font (like Lucida Console) are monospaced.

 

Note that there are two current fonts: normally an ANSI font is used, but if you are displaying the IBM/OEM character set an IBM/OEM font must be used.  If the active file is displaying IBM/OEM character set then you are selecting the current IBM/OEM font and you only see IBM/OEM fonts in the font list, otherwise you are selecting the current ANSI font.

 

A quick way to change the font is to use the Font_tool.

Font style

Type the name of a style or select from the list of styles supported by the current font. Most fonts support the regular style plus bold and italic. Other styles such as light, heavy etc are possible.

Size

Type a point size or select one from the list. Truetype fonts are scalable to any size; for other fonts only use values that are displayed in the list of available sizes.

 

A quick way to change the font size is to use the Toolbar buttons for the Increase Font Size command and the Decrease Font Size command or use the Font Size tool.

Script

Lists the available language scripts for the specified font. Pick the one appropriate for the language your computer is set up for.

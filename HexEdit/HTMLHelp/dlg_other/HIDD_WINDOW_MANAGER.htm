Window Manager Dialog

 

 

This dialog allows you to organize the currently open windows. Select one or more windows in the list on the left, then use the buttons on the right to operate on the selected window(s).

 

You can activate a window by double-clicking on it, or by selecting it and clicking the Activate button. You can also save close or minimise one or more windows by selecting them and clicking the appropriate button.

 

You can cascade or tile (vertically or horizontally) 2 or more windows by selecting them and choosing the appropriate button. Those window that are not selected are minimised.

 

Select Window

Displays a list of windows open in HexEdit. Click a window to select it. Use Shift-click to select more than one window. Use Ctrl+click to select individual windows.

Activate

Make the window selected in the list the active window. This option is only available when there is exactly one window selected.

OK

Click the OK button to close the dialog.

Save

Click this button to save the files in all the selected windows.

Close Windows

Click this button to close all the selected windows. If a window is the last window on a file and the file has been modified you are prompted if you want to save the file.

Cascade

Arranges the selected windows in a row behind each other. All other windows are minimised.

Tile Horizontally

Moves and resizes the selected windows so they are completely visible in the display. An attempt is made to stack windows on top of each other. Non-selected windows are minimised.

Tile Vertically

Moves and resizes the selected windows so they are completely visible in the display. An attempt is made to stack windows next to each other. Non-selected windows are minimised.

Minimize

All select windows are minimised. Non-selected windows are not affected.

Open Special dialog

 

 

Use this dialog to open a disk for viewing or editing.  These are divided into Logical Volumes, which are the mounted filesystems, and Physical Drives which are the raw drives, including hard disks, floppy drives, optical drives (CD, DVD etc), removable drives (flash drives, Zip drives, hard disks etc) - basically anything that can store data and looks like a "drive".

 

It is very dangerous to modify volumes or drives unless you know what you are doing.  Even then you should have a backup in case of mistakes.

 

Disk

Select a disk from the list and click the OK button to open it.

 

Under logical volumes are all the currently mounted file systems (which typically have an associated drive letter shown after the name).  These do not always correspond one-to-one with physical devices - for example a hard disk may have multiple partitions which are mounted individually.  Only file systems supported by the operating system are shown - for Windows 9X (95, 98, ME) these are FAT type file systems only.  NT only supports NTFS file systems whereas Windows 2000 and later (ie XP, Vista, etc) supports both FAT and NTFS.  Also optical media filesystems are also supported (CDFS etc).

 

Physical drives are the raw disk drives.  (Note that physical drives are not supported under the Windows 9X operating systems.)  For any logical volume above there is also a physical drive that contains the volume's data.  When possible the drive letter associated with the device is shown - but this is not always possible, for example, a hard disk may have several partitions all with their own drive letter.

Properties

When opening drives it is often difficult to know (in other disk editors) exactly which device name corresponds to the device you want to work with, without hunting through device manager or even taking the cover off your computer to check the manufacturer and model.  For this reason HexEdit provides as much information as it can (drive letters, drive capacity, free space etc) about each disk so you can be confident you are using the right one.

 

For logical volumes the file system type (eg, NTFS or FAT32) and cluster size are shown as well as the capacity and free space in three ways (abbreviated, total bytes, total clusters).

 

For physical drives the manufacturer, model number, serial number, EPROM revision etc are shown (if available), as well as the nominal physical layout (sectors, cylinders etc) and the total capacity.

Open as read only

Unless you are changing something open the device read-only for safety.  The checkbox is disabled (and checked) for read-only devices (eg CD) or media (eg write protected floppy) since you can't modify them.

OK

Click OK to open the selected device.  This button is disabled if there is no device selected or it cannot be opened for some reason, such as no medium present for a removable device.

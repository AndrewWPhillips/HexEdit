Colour Scheme tool                              

 

 

 

Use this tool to select a different colour scheme for the active file.

 

To change the colours of a colour scheme or add and delete colour schemes use the Colour Options page of the Options dialog.

 

Note that if you use one of the four standard colour schemes (ASCII default etc) then when you change between different character set displays the colour scheme will automatically change.  For example, if the current character set is ASCII (see Char Set ASCII) and the current scheme is "ASCII default" then changing to the EBCDIC character set (see Char Set EBCDIC) automatically changes to the "EBCDIC default" colour scheme.

How to get help

 

 

HexEdit Pro contains extensive on-line help.  For example, you can obtain useful help on any command by holding the mouse over a menu item or toolbar button and pressing the F1 key.

 

Most dialog windows contain a Help button.  This provides information on the purpose of the dialog and how to use the controls within it.

 

Dialogs also contain a small button containing a question mark near their top right corner.  Click this button and then click on a control within the window to see a brief description or what the control is for.  You can also right-click a control and select "What's This" from the popup menu to see the same description.

 

You can access the full on-line help system using the "Help Topics" item on the Help menu or by pressing F1.  Also pressing Shift-F1 places HexEdit Pro into help mode -- click any object, such as a toolbar button, to find out about it.

 

See the Help topic in the Overview of HexEdit Pro for more information.

 

Overview

 

What is HexEdit Pro?

HexEdit Pro is a binary file editor.  It is used to view and modify binary files.

Who uses HexEdit Pro?

During testing and debugging programmers find HexEdit Pro invaluable for viewing and fixing binary files written or read by their programs.  Power users can use HexEdit Pro to patch files or fix corrupt files such as documents and databases.

Benefits

HexEdit Pro saves time by allowing you to track down and fix problems.  The intuitive user interface means you can work quickly to accomplish tasks that are difficult in other hex editors.

Powerful features such as keystroke macros mean you can easily do things that might otherwise take hours or days to write a program to do the same thing. Templates, configurable color schemes, property pages and search/compare tools make it easy to find what you are looking for.  Numerous user-interface refinements make HexEdit Pro a joy to use.

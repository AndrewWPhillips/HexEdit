Explorer window

 

 

 

The Explorer window is a HexEdit Pro modeless dialog that resembles a Windows Explorer window but has several special features for use within HexEdit Pro, namely:

  • Double-click a file in the file view to open it in HexEdit Pro.

  • Right-click a file to display the Explorer context menu with 2 additional items allowing you to open the file or open it read-only.

  • Enter file type filter(s) to see a subset of files.

  • You can choose filters from your own set of custom filters, or from the history list of previously entered filters.

  • Type a path into the address box and press Enter to display the contents, or select from the history list of previously entered ones.

  • Easily toggle the display of hidden files.

  • Toggle between having folder and files views arranged vertically or horizontally to suit you screen layout and dimensions.

  • Display columns specific to HexEdit Pro such as time last opened, comments etc.

  • Show or hide columns b clicking any header and choosing from the list

Back Button

Takes you back to the previous folder visited.

Forward Button

Takes you forward to the next folder if you have use the above Back button..

Parent Button

Takes you to the parent folder - ie, up one level in the directory tree.

Filter

Restricts the files are shown in the File View window.  Only files that match the filter pattern are displayed.  Type in a filter or select one from the predefined filters list using the button to the right.

Predefined Filters Button.

HexEdit Pro allows you to set up your own list of custom filters using the File Type Filters page of the Options dialog.  Click the right-arrow button to select from the list of custom filters or use the bottom entry to edit the list

Views Button

Shows a drop-down menu with options on how to view the files.  The set of options depends on the version of Windows you are using but includes icon and details views.  Details view provides the most information including several resizable columns that display such information as file size, last modification time, attributes etc.

 

Also shows options to turn on or off the display of hidden files (files with the hidden attribute set).

Orientation button

This button toggles between having the folders/files panes positioned vertically or horizontally.

Address

This displays and allows you to enter the full path of the folder being displayed.  A drop-down history list is available using the down-arrow button.

Refresh Button

Click this button to refresh the folder and file views if you think something may have changed (file added, deleted, modified etc).

Folders View

The left ot top pane shows all the folders in the local file system in a hierarchical (tree) view.  Click on any folder to show its contents in the adjacent file view.

File View

The right or bottom pane shows the files and folders in the current folder.

 

You can click a column header to sort by that column.  Drag a column to change the order of the columns.  Drag the separator between columns to resize them.

 

Right-click any column header to see a list of columns you can display or hide.

Operations menu commands

 

The Operations menu contains commands that operate on the file:

 

Convert ASCII->EBCDIC

Convert all bytes of the selection from ASCII to EBCDIC.

Convert EBCDIC->ASCII

Convert all bytes of the selection from EBCDIC to ASCII.

Convert ANSI->IBM/OEM

Convert all bytes of the selection from ANSI to IBM/OEM.

Convert IBM/OEM-ANSI

Convert all bytes of the selection from IBM/OEM to ANSI.

Convert to Lowercase

Convert upper case characters to lower case in current character set.

Convert to Uppercase

Convert lower case characters to upper case in current character set.

Encryption Algorithm

Sets the encryption algorithm to use

Encryption Password

Sets the password for subsequent encryption/decryption

Clear Password

Clears the current encryption password

Encrypt

Encrypts the selected bytes

Decrypt

Decrypts the selected bytes

Compression Settings

Adjust ZLIB settings used for compression (and decompression).

Compress

Compress the selection using ZLIB algorithm and current settings.

Decompress

Decompress the selection assuming it has been compressed using ZLIB.

Generate Fast Random Numbers

Replaces selected bytes with random bytes using a fast algorithm

Generate Good Random Numbers

Replaces selected bytes with good random bytes.

Seed from Calculator

Seeds random numbers generators with the current calculator value.

Random Seed

Seeds random number generator using the current time.

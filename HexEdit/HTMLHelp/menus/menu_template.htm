Template menu commands

 

The Template menu offers commands for use with templates:

 

New

Start new template.  Discards the current template and loads "Default" template.

Open File Type

Loads a template associated with a file extension.

Open Other

Loads a template that is not associated with a file extension.

Save

Saves the current template. Prompts for a file name if "Default" is the current template.

Save As

Saves the current templates to a different file name.

Hide Tree View

Hides the template for the active file.

Split Window

Displays the current template in a split window.

Tabbed Window

Displays the current template using tabs at the bottom of the window.

Synchronise Tree and Data

Synchronizes the selection in the template view and the hex view.

Auto Sync Tree/Data Toggle

Toggle automatic synchronization of the template view and the hex view.

Refresh Tree View

If the file has changed it recalculates the template based on the new file contents.

Allow Editing

Toggles template editing.

Settings

Displays global settings associated with the template.

Template Web Site

Opens web browser at the web site associated with the current template.

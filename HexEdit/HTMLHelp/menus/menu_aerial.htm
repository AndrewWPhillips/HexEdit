Aerial menu commands

 

The Aerial menu offers commands to show and manipulate the aerial view window:

 

Hide Aerial View

Close the aerial view

Split Aerial View

Show the aerial view in a split window

Tabbed Aerial View

Show the aerial view in a tabbed window

Show Selection in Aerial Border

Indicates the selection location in aerial view top and left borders

Show Mark in Aerial View Border

Indicates the position of the mark in the aerial view left border

Show Highlights in Aerial Border

Indicates highlights in the aerial view left border

Search Occurrences in Aerial Border

Indicates search occurrences in the aerial view left border

Show Bookmarks in Aerial Border

Indicates bookmarks in the aerial view left border

Marching Ants: Selection

Indicates the selection in aerial view using black marching ants

Marching Ants: Mark

Indicates the mark in aerial view using marching ants

Marching Ants: Highlights

Indicates highlights in aerial view using marching ants

Marching Ants: Search Occurrences

Indicates search occurrences  in aerial view using marching ants

Marching Ants: Bookmarks

Indicates bookmarks in aerial view using marching ants

Zoom Out Aerial View

No zoom = each byte (or group) is show as one pixel on the screen

Zoom X 2

Each byte (or group) is show as 2 X 2 pixels on the screen

Zoom X 4

Each byte (or group) is show as 4 X 4 pixels on the screen

Zoom X 8

Each byte (or group) is show as 8 X 8 pixels on the screen

Zoom X 12

Each byte (or group) is show as 12 X 12 pixels on the screen

Zoom X 16

Each byte (or group) is show as 16 X 16 pixels on the screen

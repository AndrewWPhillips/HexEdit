Help menu commands

 

The Help menu offers the following commands:

 

Help Topics

Offers you a Contents page and an Index to the help topics.

Context Help

Provides help on a menu item, button, part of the screen etc.

Customization Tutorial

Starts the Customization Tutorial

Keystroke Macro Tutorial

Starts the Keystroke Macro Tutorial

Basic Template Tutorial

Starts the Basic Template Tutorial

Advanced Template Tutorial

Starts the Advanced Template Tutorial

Support Forum Page

Connects to the Forums on the HexEdit web site to get the latest news and ask for help

Help via the Web

Connects to the web version of the on-line help which may include more up to date info

FAQ Web Page

Views the FAQ page on the HexEdit web site for the latest FAQ information

HexEdit Home Page

Connects to the main HexEdit web page

Registration Page

Connects to the on-line registration page to register and pay for HexEdit

Help via Email

Sends email for tech support

Restore Dialogs Command

Restores (makes undocked etc) all modeless dialogs

Repair Copy User Files

Create/restore HexEdit support files for the logged in user

Repair Customizations

Restores factory default menus, keyboard layout, toolbars etc

Repair Settings command

Removes all settings stored in the registry

Repair All command

Removes all settings, internal data files, and registration information

Keyboard Map

Display a table showing the current keyboard shortcuts.

Tip of the Day

Displays the latest “Tip of the Day”

About

Displays the version number of this application.

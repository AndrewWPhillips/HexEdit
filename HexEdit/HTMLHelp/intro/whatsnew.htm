What’s New in 3.4

 

Version 3.4 adds several new features not the least of which is navigation points.  Nav. pts make it easy to go back (or forward) to some earlier place where you were working within a file.  Many people use "undo" for this purpose but navigation points have the advantage that no changes will be undone.  Moreover, it will switch between files (while undo only works within a single file), even re-opening files if they have been closed in the meantime.  Navigation points are based on a similar feature in Visual Studio but they are better in a number of ways, such as the fact that the drop-down menu of navigation points is displayed in strict chronological order and grouped by file.

 

Also added is a much more powerful ruler with control over the positioning of numbers and ticks within the ruler (see the Workspace Display Options page of the Options dialog).  Whether the hex and/or decimal ruler is displayed depends on how addresses are displayed (both hex and decimal addresses can now be displayed at the same time).  Further, the ruler includes "markers" (similar to MSWord) that show the current row size, etc and can be dragged to adjust these settings.

 

Related to the ruler changes is the ability to show any (or all) of hex addresses, decimal addresses or line numbers in the address area (to the left of the hex area).  The current cursor position and mouse location are (optionally) highlighted in the ruler and the address area.

 

The calculator has also had a major overhaul with a new look and the ability to evaluate any expression.  You can enter expressions of any of the 5 types that HexEdit expressions support (integer, real, string, boolean and date), which is particularly useful for testing expressions to use in templates.  There are now drop down lists in the calculator to choose values from the histories of the hex and decimal jump tools, variables and over 40 pre-defined functions.  The calculator also has a drop-down history list so you can easily re-use or modify an expression you have previously entered.

 

The Options dialog has been improved with a better tree navigation system showing icons for each page for easy reference.  There are also more than 20 new options (but due to the new layout and navigation system this does not make using the Options dialog any more confusing).  New inter-page jump buttons make it easy to find related options or options that have been moved to a different page from earlier releases of HexEdit.

 

The Explorer window has been improved, notably with the addition of more columns, the ability to easily show/hide columns by right-clicking any column header and the ability to reorder columns by dragging them.

 

There are many other improvements such as fixes for non-standard (other than 96) DPI values.

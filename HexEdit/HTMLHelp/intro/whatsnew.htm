What’s New in 5.0

 

Since HexEdit 4.0 there have been many new features.  A few of these also appeared in HexEdit Pro 4.0, but most are completely new. (The shareware program HexEdit Pro has been discontinued and all of its features were incorporated into HexEdit 5.0.)

 

There are some major new features, several medium-level additions and lots of small level improvements and bug fixes.

  • major features are the new file compare view and bitmap preview

  • the character area now has comprehensive support for Windows code pages

  • other features are background processing + Windows 7 TaskBar support

  • there are also new operations for CRCs, encoding (Base64 etc) and digests (SHA etc)

  • many other new commands and options were added as well as improvements to existing features

New commands

  • Make Favorite (star button on Standard Toolbar) adds the file to the "Favorites" category
  • Display Window Tabs allows turning off of tabs to allow tiling, cascading, etc of windows
  • Many new commands for selecting a code page to use in the character area (see below)
  • New Compare menu that allows comparing two files (see below)
  • New Preview menu that shows a preview of graphics files (JPEG, BMP, PNG, etc)
  • New Base32, Base64 and Base64-URL commands to encode/decode binary data as text
  • New CRC commands (CCITT, MPEG2) and improved performance of existing CRC commands
  • New General CRC option (see below)
  • New digest commands SHA, SHA-2 (with 4 size options), SHA-3 (with 4 size options)

Windows 7 (and Later)

  • long operations now show progress in the HexEdit Task Bar Icon

  • hovering the mouse over the HexEdit icon shows a thub nail of files currently open

  • recently opened files are added to the Task Bar "Recent Files" list

  • commonly opened files are added to the Task Bar "Most Used" list

  • files in the HexEdit "Favorites" category are added to the Task Bar "Favorites" list

Options

  • new File Folders page gives many options for where the File Open/Save dialogs start

  • new Background processing adds many new options on top of background searching

  • new Explorer page adds options related to Explorer window (wipe strategy etc)

  • new Aerial View page allows you to set the maximum bitmap size used

  • new document settings for Preview and Compare views (see below)

  • new Code Page selection as well as invalid/continuation display character

  • ability to select all three character area fonts (normal/OEM/multi-byte fonts)

Status Bar

  • The current search occurrence is shown next to total occurrences in the status bar
    search occurrences are only shown if background searches are turned on
  • New differences pane shows the number of differences found (see Compare Files Feature below)

Tip Window

  • click and drag the tip window if it is in the way
  • relative position of tip window is remembered

Background Processing

  • several new options (in addition to background searches that HexEdit has always had)

  • all background processing is done in low-priority background threads

  • ability to count the occurrences of all 256 bytes values (for display in properties dialog graph)

  • calculate SHA and SHA-2 digests and CRC32 on the whole file

  • ability to turn off background processing for some types of files (network, removeable media, etc)
    this can avoid problems like excessive network traffic, display pauses, etc

Copy Formatted Dialog

  • new options for converting binary to text while placing on the clipboard
  • Renamed from Copy C/C++ Source with addition of "General" options
  • Simplified selection of general options (such as CSV)
  • new Preview display shows exactly what you will get

Recent File List

  • OK button changed to Close/Save to avoid confusion

  • fixed flicker of selection count

  • use the new Search button to quickly find a file by keyword/category/etc

  • Open Read Only checkbox

  • Open Shareable checkbox

Tool Windows (Modeless Dialogs)

  • default position of dialogs has been improved
  • easily restore layout (Help -> Repair -> Restore Dialog Layout)
  • new Calculator Tape window - shows history of all calculations
  • new Compare List shows all differences between compared files (see below)
  • improvements and bug fixes to Calculator, Bookmarks and Explorer (see below)
  • three new pages in Properties dialog

Calculator

  • integer calculations now support any precisionincluding Infinite

  • digest and CRC results are dumped into calculator for easy use in macros

  • easy selection of any radix from 2 to 36

  • easy selection of any number of bits 1 to Infinite

  • select signed or unsigned

  • display of current expression above text entry box

  • select previously entered expression to re-evaluate using drop-list

  • see complete history of calculations using Calculator Tape window

  • bottom 64 bits display is shown in red/blue to show hex/decimal mode
    (the actual colors depend on the color scheme in use)

Bookmarks

  • simpler (icon not text) buttons at the top of the window
  • in-place edit of bookmark names - simply click and type
  • quickly create a new bookmark with one click (then optionally enter a name)

Properties

  • Floating point page: now supports Real48 (as in Delphi, Turbo Pascal)

  • Time page: now supports most common time64_t (C/C++) format

  • new Counts page: graph of byte counts (see Background Processing)

  • new Stats page: CRC32, MD5, SHA, SHA-2 of whole file (Background Processing)

  • new Bitmap page shows bitmap properties if Preview is open (see below)

Explorer Window

  • custom context menu when a file is right-clicked

  • quickly set a file time based on a DATE var (or now)

  • quickly set all file times (created/modified/accessed)

  • quickly set/clear common file flags (read-only etc)

  • even set/clear SYSTEM flag bit

  • quickly set/clear compressed and encrypted flags

  • delete file(s) from disk

  • "wipe" file(s) with 3 options trading speed for security

Preview Feature

  • preview graphics format files (JPEG, PNG, BMP, etc)

  • display in split or tabbed window

  • modify bitmap in memory and instantly see the effects

  • complete control of zoom/pan using menus or mouse

  • full display for transparency (alpha channel)

  • 4 different backgrounds for viewing transparency

  • checkerboard background allows better view of partial transparency

  • new Bitmap Properties page shows bitmap info (on disk + in memory)

File Compare Feature

  • compare with a different disk file

  • other file view is shown in split or tabbed window

  • synchronous scrolling of both views

  • synchronize selection between both views

  • copy to clipboard in compare window

  • Compare List tool window lists all differences

  • menu/keyboard commands to move through differences

  • Option to find insertions/deletions

  • Option to compare a file with itself to see changes over time

  • uses SSE2 (SIMD) instructions for huge performance boost

Technical Notes

  • The Wix installer is no longer used - we now use InstallAware

  • HexEdit uses the Boost library for many things - of particular recent note is CRCs

  • Crypto++ is now also used - for digests (MD5, SHA, etc) and encoding (Base64 etc)

  • Infiinite precision integers (as used in the calculator) are now handled by MPIR library

  • SVN is no longer used for version control - we now use Git

  • HexEdit source is no longer hosted by SourceForge, instead by GitHub

  • Visual Studio 2010 or later is required as HexEdit now uses SSE2 intrinsics
    for greatly improved performance (eg, NOT operation and file compare)

What’s New in 3.3

 

Version 3.3 has had a major reorganisation of the Options dialog.  Instead of the tabs at the top of the dialog, navigation to specific pages is done using a hierarchical "tree" view on the left.  Moreover more than a dozen new options have been added or exposed.

 

A new "Explorer" window (actually a modeless dialog) has been added.  This behaves very much like Windows Explorer but has many additional features that make it a very useful feature of HexEdit.  For example, you can simply double-click a file to open it in HexEdit.  In Details view there are columns which do not appear in the normal Windows Explorer such as the time you last opened the file in HexEdit, comments you have attached to the file in the Properties dialog etc.

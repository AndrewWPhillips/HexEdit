Acknowledgments

 

Code/Libraries used in HexEdit

 

MFC from Microsoft

Microsoft Foundation Classes is a useful library which wraps the Windows API in C++ classes.  MFC comes with Visual Studio (see below).

 

Boost from various

Boost is an excellent open source C++ template library.  I first used the BGL (Boost Graph Library) in the late 1900's.  I have made use of several parts of the library in HexEdit, such as CRCs, random numbers and boost::tuple.

 

FreeImage by Hervé Drolon et al

FreeImage is a brilliant open source project for dealing with image files and manipulating images in memory.  It made possible the HexEdit Aerial View bit is also used in other places in HexEdit.  It has been ported to several platforms but we only use the Win32 DLL that they provide.

 

MPIR - by many and derived from GMP from GNU

Superb quality Multiple Precisions Integer (and Rationals) library used to replace the BigInteger class for integer calculations in the calculator. This allows for integers of more than 64-bits to be used (essential for digests) including the infinite precision option.

 

Crypto++ by Wei Dai et al

Excellent C++ library for encryption and related algorithms. Currently this is just used for digests (MD5, SHA1, SHA2 and SHA3).

 

MD5 algorithm by Ron Rivest

This algorithm is a cryptographic checksum (sometimes also called a one-way hash function or message digest). It generates a 128-bit value that is often used to check the veracity of files transferred over an insecure link in order to detect tampering.

 

ZLIB Compression library by Jean-loup Gailly and Mark Adler

This is a free general purpose compression library (with C source available) which is used extensively in the industry.  (I would consider it to be a de facto standard).  The Compression Options dialog offers complete control of all ZLIB options.

 

MFC Grid Control by Chris Maunder

An excellent MFC grid control, which is used in several places in HexEdit - for example, in the Filters page and the Info Tips page of the Options dialog, the Recent File dialog and also (with Tree Extensions - see below) the template tree view.

 

MFC Grid Tree Extensions by Ken Bertelson

An extension to the MFC grid control that allows one of the columns to act as a "tree control".  This allows the effect of a multi-column tree control which is invaluable for the HexEdit template (tree) view.

 

WFC CFile64 Class by Sam Blackburn

When I released the source code for HexEdit 1.0 in 1999, Sam enhanced it to use "64-bit" file access, thereby allowing HexEdit to easily work with files greater than 4 GBytes (2^32) in size.  (Windows supports 64 bit file positioning but the MFC CFile class only allowed 32-bit file addresses).  Sam's CFile64 class (incorporated, with modifications, into HexEdit ) is just one of numerous useful (non-UI) classes provided by his WFC library.

 

Window Splitter class by Robert A. T. Káldy

A simple, functional class that provides a nice-looking splitter.  Unlike MFC splitters I found it very easy to use and extend for my needs.  This is used for the split between the template "tree" view and/or "aerial" view and the "hex" view of a file.

 

Dialog Resize class by Herbert Menke

A simple to use but very effective class that allows automatic move/resize of controls in a dialog when it is resized.  This is used for all resizable dialogs in HexEdit such as the Recent File dialog, Bookmarks dialog etc.

 

range_set class by Andrew Phillips

I originally wrote this as a template class which emulates the STL std::set class but much more efficiently handles ranges of values, as well as adding facilities to input/output ranges as a string.  An article on it appeared in the June 1999 C/C++ Users Journal.  Since then it has found numerous uses in HexEdit such as in the color range selection (Colors page of Options dialog) or the matching range of values for a CASE of a template SWITCH statement.

 

Folder Selection class by Andrew Phillips

I, like many, find the "standard" way of selecting a directory in Windows (SHBrowseForFolder) extremely annoying.  There are numerous things wrong with it but the worst are that you can't see the files in a directory, you can't type in the path or part of it yourself, or resize the window etc etc.  So I wrote the class which extends the standard file open dialog (extending MFC's CFileDialog).  In HexEdit it is used to select the folder where macros (see Macro Options page) and templates (see Template Options) are stored.

 

Transparent Static Control by Ali Rafiee

An MFC class that provides the static controls -- used in the HexEdit bout box.

 

 

Software Used to Develop HexEdit

 

Visual Studio by Microsoft

HexEdit has always been developed using Visual Studio and the Microsoft C++ compiler that comes with it.

 

GIT source control by Linus Torvalds et al (previously Subversion source control by Jim Blandy et al)

HexEdit now uses Git for version control.

 

GitHub

HexEdit (open source) is currently hosted on GitHub at HexEdit.

 

RoboHelp by eHelp (now Adobe)

Since HexEdit 2.1 the on-line help has been provided in Html Help format.  It is created using RoboHelp Office.

 

InstallAware (previously WIX Installer Toolkit by Rob Mensching et al)

This installer works OK.

 

VMWare Workstation by VMWare

This software allows running virtual machines with different versions of Windows installed and works very well.  It makes testing HexEdit on all the different versions of Windows (95,98,98SE,ME,NT,2K,XP,Vista,W7) easy.

 

Paint.NET by Rick Brewster et al

A very useful and free piece of software.  (I especially like the CodeLab plugin.)  Paint.NET was used for the new template icons and other images.

 

Real World Icon Editor by Real World Graphics

Another very well done program.  I used it to create the icon file used in HexEdit.  Their "ReplaceVistaIcon.exe" utility was very useful for getting around deficiencies of Visual Studio 2003.

 

xplorer² lite by Nikos Bozinis

A.wonderful Windows Explorer replacement which I use for all my file management.  I use this program many times per day and it has saved me a lot of time.

 

CamStudio

A.simple but effective free (open source) screen recording  tool I use to capture videos for demonstrating HexEdit .

 

VirtualDub

A free video editing tool I use to edit the videos I capture using CamStudio.  (I have Adobe Premiere but find this tool much simpler and more effective.)

 

Sources of Information

 

Revolutionary Guide to MFC 4 Programming by Mike Blaszczak

Around 1997 I started learning MFC using this book.  (I hadn't really done any serious C++ or Windows programming at the time either.)  HexEdit started out as an experiment after fiddling with the "HexView" demo program provided with this book (which was to be honest pretty poor).

 

Undocumented Windows 2000 Secrets by Sven B. Schreiber

This book was invaluable in creating the disk editor that is provided for Windows 2000/XP/Vista.

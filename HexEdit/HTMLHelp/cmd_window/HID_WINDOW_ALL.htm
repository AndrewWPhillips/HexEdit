1, 2, … command (Window menu)

 

HexEdit displays a list of currently open windows at the bottom of the Window menu. A check mark appears in front of the name of the active window. Choose a window from this list to make its window active. You can also flip through all the open windows with the Next Window command (Ctrl+F6).

 

Shortcuts

Mouse: Click on a visible part of the desired window to activate it, or click on the tab for the window at the top or bottom of the HexEdit display if enabled (see Window Display Options page).

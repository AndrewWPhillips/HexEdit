Printing

 

If you are acquainted with printing in other Windows programs printing is fairly simple and straightforward. What you see in the window is almost exactly what is printed.  There are optional extras though, such as printing headers and footers, a border around the whole page, etc.  To see how it looks and how the file would be spread across pages you can use the Print Preview command. The preview depends on the currently selected printer which you can change with the Page Setup Command as well as options such as landscape or portrait printing etc.

 

Page Layout

 

The Printer Options page of the Options Dialog allows setting of page margins, line-spacing, header/footer etc.  There are also other options such as printing column headers so you can easily work out the exact address of any byte. The header and footer are displayed at the top and bottom of each page can be any text, including special codes for the current date/time, page number, filename, time file was last modified, as well as category, keywords and comments if you have added any using the Summary Page of the Properties dialog.

 

If you have a wide screen (or use a long line length and the scroll bars) then the printed lines may not fit across the page.  In this case a smaller version of the screen font is used.

 

Partial Printing

 

It is not unusual to want to print a small part of a binary file. You can do this in HexEdit Pro by printing a range of pages or by printing the selection.

 

To print a range of pages but you must first use Print Preview to find out the numbers of the pages you want to print.  Another problem with printing a range of pages is that the standard Print Dialog in Windows has problems if the total number of pages exceeds 32,767. If you want to print part of a very large file such as a database, this is a problem.

 

A better way to print part of a file is to select the bytes you want to print and use the “Print Selection” option of the Print Dialog.  Also when you print the selection you have the option of not printing duplicate lines (instead the line is printed just once with a count of the number of repeats underneath) -- this can be very useful if the section you want to print contains large areas of bytes all the same such as 00 or FF.

 

Template Tree View

 

To print (or print preview) the template tree view simply select an item in the tree view and use the File/Print (or File/Print Preview) command as usual.  Note that the tree view is printed exactly as displayed.  You may want to expand or contract different branches before printing.  Also the text is sized to fit across the page so you may need to resize or hide some columns if the text prints too small.

 

For further information see:

Print command to print all or part of the file

Print Preview command to preview the printed pages

Printer Options page to change header/footer, margins etc

File menu for printing commands

Print dialog to see options for printing the file or selection

Print Progress dialog to see progress while printing is in progress

Print Setup Dialog to find out how to select and configure the printer

Window Options Page to change the number of displayed columns etc

File Summary Page to set category, comments etc that can be printed in the page header/footer

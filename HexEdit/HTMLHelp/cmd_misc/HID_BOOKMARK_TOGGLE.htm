Toggle Bookmark

 

This command allows you to quickly add or delete a bookmark.  It deletes a bookmark if there is one at the current cursor position.  If there is no bookmark at the current cursor position it creates a new bookmark at that position using a prefix of "unnamed" followed by a number.

 

When an "unnamed" bookmark is created it is given a number one greater than the largest unnamed bookmark in existence (in all files, not just for the current file).  If there are no such bookmarks it is given the name "unnamed000".  The name of the new bookmark should be immediately visible in the Bookmark_tool.  Creating unnamed bookmarks this way is faster than entering a name into the Bookmarks dialog.

 

Shortcuts

Keys: Ctrl+F2

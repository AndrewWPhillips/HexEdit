Case-sensitive Search command

 

Puts you in the find tool on the edit bar ready to perform an exact text search. Any hex search that was already in the find tool is converted to text, but typing new characters overwrites it. To start the search type the text and press Enter.

 

Use the Find Forward command to search for the next occurrence of the same string.

 

Shortcuts

Mouse: Click the find tool and type an equals sign (=) to begin an exact text search

Keys: F5

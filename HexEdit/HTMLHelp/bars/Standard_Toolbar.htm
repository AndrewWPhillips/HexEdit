Standard Toolbar

 

 

By default the Standard Toolbar appears beneath the menubar.  It provides quick mouse access to several common tools in HexEdit Pro. These include facilities for file handling, using the clipboard and changing the window display.

 

Another default toolbar is called the Edit Bar, which has commands useful during editing.

 

To hide or display the Standard toolbar, choose the View Standard Toolbar from the View menu.

 

To customize this or any toolbar or even create new toolbars use the Toolbar page of the Customize dialog.

Navigation Bar

 

 

The Navigation Bar provides facilities for navigating in the current and (via bookmarks) other files.  It mainly includes the Bookmark tool which shows the closest bookmark and allows you to quickly jump to any bookmark in the current file.  There are also buttons for setting and jumping to the mark and bookmarks as well as the Hex Address tool and Decimal Address tool which also appear in the Edit Bar.

 

To hide or display the Navigation Bar, choose the View Navigation Bar command from the View menu.

 

To customize this or any toolbar or even create new toolbars use the Toolbar page of the Customize dialog.

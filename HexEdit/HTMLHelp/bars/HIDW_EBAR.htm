Edit Bar

 

 

The Edit Bar, by default, is displayed across the top of the application window, below the Tool Bar. If your screen is wide enough you may prefer to drag it next to the Tool Bar.

 

The Edit Bar is a toolbar which, like the Standard Toolbar, Navigation Bar etc provides quick mouse access to tools useful when editing files in HexEdit Pro. These include facilities for searching, jumping and marking positions.

 

To hide or display the Toolbar, choose the Edit Bar command from the View menu.

 

To customize this or any toolbar or even create new toolbars use the Toolbar page of the Customize dialog.

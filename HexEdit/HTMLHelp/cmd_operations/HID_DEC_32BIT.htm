Decrement Double Word command (Operations menu)

 

This command subtracts one from the double-word (4 bytes) after the cursor or from all double-words in the selection if there is one. This command is not available if the cursor is too close to the end of the file or the number of bytes selected is not divisible by 4.

 

The values are assumed to be in the normal Intel format with the least significant byte first (little-endian). However, to increment using big-endian format use the Flip Double Word command before and after this command. Zero wraps around to the maximum value (2^32 - 1) when decremented.

 

Also see the Increment Double Word command.

Increment Double Word command (Operations menu)

 

This command adds one to the double-word (4 bytes) after the cursor or to all double-words in the selection if there is one. This command is not available if the cursor is too close to the end of the file or the number of bytes selected is not divisible by 4.

 

The values are assumed to be in the normal Intel format with the least significant byte first (little-endian). However, to increment using big-endian format use the Flip Double Word command before and after this command. The maximum value (2^32 – 1) wraps around to zero when incremented.

 

Also see the Decrement Double Word command.

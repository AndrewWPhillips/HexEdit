Subtract From Double Word Command (Operations menu)

 

This command subtracts the double word at the cursor position from the current value in the calculator and saves the value back to the file.  If there is a selection then it operates on all double word of the selection.

 

This command is not available if there is no active window or the cursor is positioned at the end of the file.  Further, the number of bits in the calculator must be set to 8 bits, since this command only operates on bytes.

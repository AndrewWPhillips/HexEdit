Seed RNG from Calculator (Operations menu)

 

This command seeds the PRNGs (pseudo random number generators) using the current value in the calculator. All random number generators used in HexEdit are seeded by this command.

 

To obtain the same sequence of random numbers use the same seed value.  Using the same sequence of numbers is useful in situations where results need to be reproducible (eg during debugging).  However, typically you would use a random seed using the Random Seed command

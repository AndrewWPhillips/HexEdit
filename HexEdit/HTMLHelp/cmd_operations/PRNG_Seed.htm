Seed Random Number Generator (Operations menu)

 

This command seeds the PRNGs (pseudo random number generators) using a value that should be somewhat random since it is based on milliseconds of the current time. There are two PRNG algorithms available in HexEdit and both are seeded by this command.

 

This command is provided as a convenience.  The randomness of the seed value is not guaranteed and maybe somewhat predictable.  For best results place a truly random number in the calculator and use the Seed Random Number Generator from Calculator command.

 

Note that this command should only be called once before generating a sequence of random numbers.  Reseeding before generating each number will produce numbers that are not very good random numbers, since they will be related to the current time and probably not very good statistically either.

 

To obtain the same sequence of random numbers load the same specific value into the calculator and use the Seed Random Number Generator from Calculator command.

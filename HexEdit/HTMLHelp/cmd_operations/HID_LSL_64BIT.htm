LSL Quad Word Command (Operations menu)

 

This command performs a LSL (shift left) operation on the current quad words by the number of bits specified in the calculator.  New (right-most) bits are filled with zero so if the calculator has a value of 64 or more then the quad word is just set to zero.  If there is a selection then it operates on all quad words of the selection.

 

This command is not available if the number of bytes selected is not a multiple of 8, there is no active window or the cursor is too close to the end of the file.

 

Also see the Logical Shift Right 64 Bit Command.

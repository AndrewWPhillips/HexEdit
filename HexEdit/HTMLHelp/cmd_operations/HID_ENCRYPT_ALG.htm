Encryption Algorithm (Operations menu)

 

This command invokes the Encryption Algorithm Dialog to choose the cipher used with subsequent use of the Encrypt and Decrypt commands.  The algorithms available are all those provided by the Cryptographic Service Providers that are installed on your computer, as well as a bonus algorithm that comes with HexEdit.  See Encryption for more details.

 

If you use the HexEdit internal algorithm the selection must be a multiple of the block length (8 bytes) before you can encrypt or decrypt.  If you use a CSP block algorithm the encrypted length will be increased slightly, but decreased again when you decrypt -- if you are in overtype (OVR) mode HexEdit will prompt you to turn on insert (INS) mode.  If you use a CSP stream cipher then you can encrypt a selection of any length and the size remains unchanged.

 

Always decrypt using the same CSP, algorithm and password that you encrypted with.  If you don't encrypt the whole file you also need to remember the length of the encrypted data.  If you decrypt on a different machine you must make sure that the same CSP that was used to encrypt is available to decrypt.

 

Shortcuts

Less (Signed) Quad Word command (Operations menu)

 

This command compares the quad wordat the cursor position to the value in the calculator and replaces it if the value in the calculator is smaller.  Hence you can use this command to ensure that all values are set to be at most a maximum value.  Note that the comparison is done using signed (2's complement) arithmetic.

 

If there is a selection then it operates on all quad words of the selection.

 

This command is not available if the number of bytes selected is not a multiple of 8, there is no active window or the cursor is too close to the end of the file.

 

Also see the Greater 64 Bit Command.

Greater (Signed) Double Word command (Operations menu)

 

This command compares the double word at the cursor position to the value in the calculator and replaces it if the value in the calculator is larger.  Hence you can use this command to ensure that all values are set to be at least a minimum value.  Note that the comparison is done using signed (2's complement) arithmetic.

 

If there is a selection then it operates on all double words of the selection.

 

This command is not available if the number of bytes selected is not a multiple of 4, there is no active window or the cursor is too close to the end of the file.  Further, the number of bits in the calculator must not be set to 64 bits, since this command only operates on 32 bit values.

 

Also see the Less 32 Bit Command.

XOR Quad Word Command (Operations menu)

 

This command XORs the quad word at the cursor position with the current value in the calculator.  If there is a selection then it operates on all quad words of the selection.  XORing the with the same value twice returns the original value.

 

This command is not available if the number of bytes selected is not a multiple of 8, there is no active window or the cursor is too close to the end of the file.

8 Bit Checksum (Operations menu)

 

This calculates the checksum of all the bytes selected and stores the 8-bit result in the calculator.  If there is no selection the command is not available.

 

This command calculates a checksum by adding all the bytes, and throwing away any carry.  Sometimes a different "checksum" is used which is the value added to the bytes so that all the bytes add up to zero -- for this type of checksum just negate the value (using the +/- button) in the calculator once you have called this command.

 

A checksum is a simple method of checking that all the bits of a set of bytes have not changed.  However, it is fairly easy for compensating errors to occur which generate the same checksum.  A better result is obtained by using a checksum with more bits.  An even better way to check for this sort of error is to use a CRC (see CRC 32 and CRC CCITT).

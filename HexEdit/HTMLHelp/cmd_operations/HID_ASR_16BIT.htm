ASR Word Command (Operations menu)

 

This command performs an ASR (arithmetic shift right) operation on the current word by the number of bits specified in the calculator.  If the calculator has a value of 16 or more then the word is just set to 0000 or FFFF.  If there is a selection then it operates on all words of the selection.

 

The operation effectively divides a signed (2s-complement) integer by a power of two.  New (left-most) bits are filled with same top-most bit (sign-extension).  The corresponding LSR command also shifts right but always fills with zeroes, effectively dividing an unsigned integer by a power of two.

 

This command is not available if an odd number of bytes are selected, there is no active window or the cursor is too close to the end of the file.

 

Also see the Shift Left 16 Bit Command and the Logical Shift Right 16 Bit Command.

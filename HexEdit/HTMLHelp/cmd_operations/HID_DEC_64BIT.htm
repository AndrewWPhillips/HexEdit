Decrement Quad Word command (Operations menu)

 

This command subtracts one from the quad-word (8 bytes) after the cursor or from all quad-words in the selection if there is one. This command is not available if the cursor is too close to the end of the file or the number of bytes selected is not divisible by 8.

 

The values are assumed to be in the normal Intel format with the least significant byte first (little-endian). However, to increment using big-endian format use the Flip Quad Word command before and after this command. Zero wraps around to the maximum value (2^64 - 1) when decremented.

 

Also see the Increment Quad Word command.

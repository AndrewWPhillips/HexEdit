32 Bit Checksum (Operations menu)

 

This calculates the checksum of all the double-words selected and stores the 32 bit result in the calculator.  If the calculator is in 8- or 16-bit mode then it is changed to 32-bit mode to hold the result.  If there is no selection or the size is not divisible by 4 then the command is not available.

 

This command calculates a checksum by adding all the double-words, and throwing away any carry.  Sometimes a different "checksum" is used which is just the value added to the others so that all the double-words add up to zero -- for this type of checksum just negate (using the +/- button) the value in the calculator once you have called this command.

 

A checksum is a simple method of checking that all the bits of the double-words have not changed.  However, it is fairly easy for compensating errors to occur which generate the same checksum.  A better result is obtained by using a checksum with more bits.  An even  better way to check for this sort of error is to use a CRC (see CRC 32 and CRC CCITT).

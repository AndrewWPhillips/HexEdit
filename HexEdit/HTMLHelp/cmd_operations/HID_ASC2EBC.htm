Convert ASCII to EBCDIC command

 

This command converts all bytes in the current selection from ASCII to EBCDIC. Any values that cannot be represented in EBCDIC are converted to a zero byte value.

 

You can also convert to EBCDIC using the Paste As EBCDIC command. However, the behaviour of this command is slightly different in that invalid characters are removed, whereas the Convert ASCII to EBCDIC command will never change the length of the file.

 

Also see the Convert EBCDIC to ASCII command.  Use the EBCDIC Characters command to toggle the display of characters between EBCDIC and ASCII.  This also affects how characters are copied to the clipboard.

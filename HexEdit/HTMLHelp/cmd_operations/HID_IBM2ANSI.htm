Convert IBM/OEM to ANSI command

 

This command converts all bytes in the current selection from the ANSI character set to the IBM/OEM character set. Note that as both of these sets are extensions to ASCII that any bytes with the high bit off (ie less than 128) remain unchanged. Any values that cannot be represented are converted to a zero byte value.

 

The ANSI extensions to ASCII consist mainly of characters used in western European languages. The IBM/OEM characters are those used in MSDOS programs (particuarly games) and consist mainly of graphics characters.

 

Also see the Convert ANSI to IBM/OEM command.  The IBM/OEM Graphics command toggles between displaying ANSI and IBM/OEM graphics characters.

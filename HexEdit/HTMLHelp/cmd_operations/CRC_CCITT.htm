CRC CCITT (Operations menu)

 

There are three variations of the CCITT CRC command provided which use slightly different CRC parameters.  They calculate the CRC of all the bytes selected and store the 16-bit result in the calculator.  If the calculator is in 8-bit mode then it is changed to 16-bit mode to hold the result.  If there is no selection the command is not available.  (The selection can be an odd number of bytes in size.)

 

Note that there is some confusion about what is the correct CRC CCITT.  You may need to try all three to get the results you expect.  See General CRC Dialog for the settings used for each of these CRCs.

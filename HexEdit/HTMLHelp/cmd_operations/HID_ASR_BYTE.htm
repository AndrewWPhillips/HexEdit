ASR Byte Command (Operations menu)

 

This command performs a ASR (arithmetic shift right) operation on the byte by the number of bits specified in the calculator.  If the calculator has a value of 8 or more then the byte is just set to 00 or FF.  If there is a selection then it operates on all bytes of the selection.

 

The operation effectively divides a signed (2s-complement) integer by a power of two.  New (left-most) bits are filled with same top-most bit (sign-extension).  The corresponding LSR command also shifts right but always fills with zeroes, effectively dividing an unsigned integer by a power of two.

 

This command is not available if there is no active window or the cursor is positioned at the end of the file.

 

Also see the Shift Left Byte Command and the Logical Shift Right Byte Command.

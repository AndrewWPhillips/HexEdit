Marching Ants- Search Occurrences (Aerial View)

 

When the aerial view is displayed search occurrences are traced with "marching ants".  These are all occurrences of the current search test when "background searches" are on (see Workspace Editing Options).

 

The marching ants consist of a slowly moving dashed line around the bytes of all found occurrences.  The colour of the dashed line is from the current color scheme (see Color Options page) - default is green.

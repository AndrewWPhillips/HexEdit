Context Menus

 

There are eight context menus that appear when you right click on certain parts of the HexEdit display.  All context menus except for the toolbars menu can be customized using the Customize Menu page of the Customization dialog.  The tables below show the default menus, before customization.  Most of these commands are also available on either the Edit or View menu.

 

Note that if you right-click a bookmark or highlight but the byte clicked is also selected then the Selection Context Menu is shown in preference to the Bookmark or Highlight Context Menu.

 

Aerial View Context Menu

 

Right-clicking on the Aerial View image invokes this menu:

 

Show Selection in Aerial View Border

Indicate the selection area in left and top borders of the aerial view

Show Mark in Aerial View Border

Indicate the position of the mark in the left border of the aerial view

Show Highlights in Aerial View Border

Indicate highlights in the left border of the aerial view

Show Search Occurrences in Border

Indicate background search occurrences in the left border of the aerial view

Show Bookmarks in Aerial View Border

Indicate bookmarks in the left border of the aerial view

Marching Ants Selection in Aerial View

Show the selection with black and white marching ants

Marching Ants around Mark

Indicate the position of the mark with marching ants

Marching Ants around Highlights

Indicate the position of highlights with marching ants

Marching Ants around Search Occurrences

Indicate the position of search occurrences with marching ants

Marching Ants around Bookmarks

Indicate the position of bookmarks with marching ants

Zoom Out Aerial View

Zoom out the aerial view (one bitmap pixel = 1 screen pixel)

Zoom X 2

Zoom where one bitmap pixel = 2 X 2 screen pixels

Zoom X 4

Zoom where one bitmap pixel = 4 X 4 screen pixels

Zoom X 8

Zoom where one bitmap pixel = 8 X 8 screen pixels

Zoom X 12

Zoom where one bitmap pixel = 12 X 12 screen pixels

Zoom X 16

Zoom where one bitmap pixel = 16 X 16 screen pixels

Customize command

Invokes the Customize dialog.

Options command

Invokes the Window Display page of the Options dialog.

Properties Command

Invokes the Properties dialog.

 

Selection Context Menu

 

Right-clicking on the currently selected bytes (either in the hex or character areas) invokes this menu:

 

Cut command

Deletes data from the file and moves it to the clipboard.

Copy command

Copies data from the file to the clipboard.

Paste command

Pastes from the clipboard replacing the current selection.

Write Selection

Write the current selection to a file which you specify.

Append Selection

Append the current selection to a file which you specify.

Append Same File

Append the current selection to the last file written or appended to.

Find Selection

Search for the next occurrence of the currently selected bytes.

Calculate Using Selection

Invoke calculator, loading the current selection.

Highlight

Highlight the current selection.

Customize command

Invokes the Customize dialog.

Options command

Invokes the Window Display page of the Options dialog.

Properties Command

Invokes the Properties dialog.

 

Bookmarks Context Menu

 

Right-clicking on any bookmark (either in the hex or character areas) invokes this menu:

 

Edit Bookmarks

Invokes the Bookmarks dialog to add or jump to a bookmark.

Clear All Bookmarks

Deletes all bookmarks in the current file.

Previous Bookmark

Jumps to the previous bookmark in the file.

Next Bookmark

Jumps to the next bookmark in the file.

Hide Bookmarks

Hides (but does not delete) the bookmarks in the current file.

Customize command

Invokes the Customize dialog.

Options command

Invokes the Window Display page of the Options dialog.

Properties Command

Invokes the Properties dialog.

Note that if you right-click a bookmark and the byte is also selected then the Selection Context Menu is shown in preference to the Bookmark Context Menu.

 

Highlight Context Menu

 

Right-clicking on any highlighted section (either in the hex or character areas) invokes this menu:

 

Highlight

Highlight the current selection or toggle highlighter mode.

Clear All Highlights

Remove all highlights and turn off highlighter mode.

Previous Highlight

Jump to the previous highlight in the file.

Next Highlight

Jump to the next highlight in the file.

Hide Highlights

Hides (but does not delete) the highlights in the current file.

Customize command

Invokes the Customize dialog.

Options command

Invokes the Window Display page of the Options dialog.

Properties Command

Invokes the Properties dialog.

Note that if you right-click a highlight but the byte clicked is also selected or bookmarked then a menu above is shown in preference to the Highlight Menu.

 

Hex Area Context Menu

 

Right-clicking in the hex area (on a byte that is not selected, highlighted or bookmarked) invokes this menu:

 

Character Display

Toggle character area display on or off.

Font command

Invoke the Font dialog to select the face, size etc of the display font.

Auto Fit command

Automatically change the number of columns to fit across the window.

Mark Position

Set the Mark to the current cursor position.

Allow Changes Toggle

Toggles between allowing changes and read-only mode.

Insert Mode Toggle

Toggles between Insert and Overtype mode.

Customize command

Invokes the Customize dialog.

Options command

Invokes the Window Display page of the Options dialog.

Properties Command

Invokes the Properties dialog.

 

Character Area Context Menu

 

Right-clicking anywhere in the character area (on a byte that is not selected, highlighted or bookmarked) invokes this menu:

 

Character Display

Turn off character area display.

EBCDIC Characters

Toggle between display of ASCII and EBCDIC characters.

Control Characters

Toggle the display of control characters (in ASCII mode only).

Graphic Characters

Toggle the display of graphics characters (in ASCII mode only).

IBM/OEM Graphics

Toggle between the display of ANSI and IBM/OEM graphics characters.

Allow Changes Toggle

Toggles between allowing changes and read-only mode.

Insert Mode Toggle

Toggles between Insert and Overtype mode.

Customize command

Invokes the Customize dialog.

Options command

Invokes the Window Display page of the Options dialog.

Properties Command

Invokes the Properties dialog.

 

Address Area Context Menu

 

Right-clicking in the address area (to the left of the hex area) invokes this menu:

 

Decimal Addresses

Toggles between displaying addresses in hex or decimal.

Go To command

Invoke the calculator to jump to an address.

Find command

Invoke the Find dialog to search using hex, ASCII, EBCDIC etc.

Set Mark Position

Set the Mark to the current cursor position.

Go To Mark

Returns the cursor to the last marked location.

Customize command

Invokes the Customize dialog.

Options command

Invokes the Window Display page of the Options dialog.

Properties Command

Invokes the Properties dialog.

 

Toolbars Context Menu

 

Right clicking on a tool bar displays a context menu that allows you to hide or show any of the tool bars or invoke the Customize dialog in order to create or change the toolbars. These options are also available on the View menu.

 

Standard Toolbar

Shows or hides the Standard Toolbar.

Edit Bar

Shows or hides the Edit Bar.

Format Bar

Shows or hides a toolbar with formatting options.

Navigation Bar

Shows or hides a toolbar with navigation options.

View Custom Toolbar

Shows or hides a custom toolbar.

Customize command

Used to create and customize toolbars

Find command

Allow Changes Toggle

Insert Mode Toggle

Customize command

Invokes the Customize dialog.

Options command

Invokes the Window Display page of the Options dialog.

Properties Command

Invokes the Properties dialog.

 

Tabs Area Context Menu

 

Right-clicking on a file tabs at the top or bottom of the file display area invokes this menu:

 

File Close command

Closes the file associated with the tab

File Save command

Saves the file associated with the tab

File Save As command

Allows you to save the file to a different file name

File Print command

Print the file

File Print Preview command

Preview the printed file

Show Icon

Toggle the display of icons on all the tabs

Tabs at Bottom

Toggle between displaying the file tabs at the bottom or top of the screen

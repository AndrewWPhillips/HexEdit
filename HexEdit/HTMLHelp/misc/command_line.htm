Command Line Options

 

If you fire up HexEdit Pro from a command line interpreter you can specify one or more files on the command line for HexEdit Pro to open. The filenames specified may include wild-cards. For example, “HexEditPro  *.BMP  *.GIF” opens all the .BMP and .GIF files in the current directory.

 

HexEdit Pro also supports the /P and /PT command line options. The Windows shell uses these to support drag and drop printing. The /P option takes one parameter – a filename – and prints that file to the default printer. The /PT option allows printing to any printer. It is followed by 4 parameters: filename, printer, driver, and port.

Append Same File (Edit menu)

 

This is the same as the Append Selection to File command, except that it appends to the same file as the last append or the file used in the last Write Selection command. That is, you are not prompted to select a file to append to. This command is unavailable if you have yet to select an append file or if there is no data currently selected.

 

This command is very powerful, allowing you to quickly build a new file from blocks of existing files. But with the power comes some risk.  The block is written immediately to the file so there is no way to abort the command or undo it, if it’s used by mistake.

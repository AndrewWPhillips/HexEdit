Write Selection to File (Edit menu)

 

Use this command to write the selected bytes to a disk file. This command is unavailable if there is no data currently selected.

 

This command brings up the standard Save As dialog which allows you to save the file using any valid file name. If the file already exists you are prompted to overwrite it. Note that if you choose to overwrite an existing file no backup copy is created even if backups are on. If you want to append to an existing file (and not overwrite any existing data) use the Append Selection to File command instead.

 

You can use this command and the Append Selection command as an alternative to cutting and pasting large blocks of memory. This avoids problems due to virtual memory exhaustion and the Windows 16 Mbyte clipboard limit.

 

This command is in some ways the inverse of the Read File command.

Highlight (Edit menu)

 

This command is used for highlighting a part or parts of a file for later reference.  The behaviour of this command depends on whether there are bytes currently selected.  If there are no bytes selected HexEdit enters highlighter mode whence all subsequent selections are highlighted until highlighter mode is turned off again.

 

If there is a selection then it is highlighted and highlighter mode is not turned on.  If the selection includes any bytes that are not highlighted then all bytes are highlighted after this command is invoked.  But if all the selected bytes are already highlighted the highlighting is turned off on them.

 

When a file is reopened (as long as it is in the Recent File List) the highlights from the previous use will be restored.

 

Highlighter mode is useful for highlighting the differences between 2 files when used with the Compare Windows command.

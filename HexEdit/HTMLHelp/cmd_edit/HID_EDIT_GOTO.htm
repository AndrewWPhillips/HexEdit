Go To command

 

This command invokes the calculator to quickly jump to a new address. To jump to a new address just enter it into the calculator numeric entry box and press the Enter key. The base used (hex or decimal) depends on the current address display (see the Decimal Addresses command), but you can change to any base and even enter the address in octal or binary.

 

You can use the calculator to jump to a relative or calculated address.  But if you just want to jump to a known absolute address it is quicker to type it into the Hex Address or Decimal Address tool (on the Edit Bar by default) and press Enter.

 

When the Go To command is invoked the calculator displays the current cursor address. This allows quick calculations of relative addresses. For example, to jump to an address that is 9 bytes forward in the file: invoke the Go To command, click the + and 9 buttons and then GO (or type the + and 9 keys then press the Enter key).

 

See the Calculator for more information on performing calculations.

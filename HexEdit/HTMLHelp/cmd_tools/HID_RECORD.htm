Record/Stop recording command (Tools menu)

 

Use this command to start or stop recording a keystroke macro.

 

You cannot use the Play Macro command while recording, but you can nest macros by playing a saved macro (using the Play command) while recording a macro.

 

If you accidentally start recording you can avoid loosing the previous macro by immediately turning off recording before pressing any keys. This also means that you can’t erase the current macro except by replacing it with something innocuous.

 

If you want to display information or instructions to the user when the macro is played back use the Macro Message command.

 

Note that settings changed while in the Options dialog are not recorded except for some in the Window page.  However, changing display options are better recorded by using View Menu Commands or the Standard Toolbar.

 

Shortcuts

Toolbar: 

Mouse: Double click the REC indicator on the status bar

Keys: F7

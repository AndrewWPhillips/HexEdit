Play command (Tools menu)

 

Use this command to play the current macro or a saved macro several times repeatedly.  When recording a macro you can also use this to record a call to a nested macro.

 

The command brings up the Replay Macro dialog that allows you to use the current macro or select a saved macro (ie a macro previously saved using the Save Current Macro button of the Macro Options page) from the list.  You can also set the number of times to play the macro.

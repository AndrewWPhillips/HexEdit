Control Characters: C/C++ command (View menu)

 

This command displays control characters in the character area as their C/C++ escape sequence as defined by the ISO C standard.  This command is not available if the character area is not displayed.  It is also not used when the IBM/OEM or the EBCDIC character set is displayed.

 

For example, a carriage return is displayed as 'r', a tab as 't', etc.  Control characters without a C/C++ escape sequence are displayed as a dot.  A byte with value zero is displayed as '0'.  They are usually differentiated by from normal ASCII characters by being displayed in red, although this depends on the color scheme in use (see Color Options page).

 

Alternative ways to display control characters are provided by the Control Chars None, and Control Chars Alpha commands.

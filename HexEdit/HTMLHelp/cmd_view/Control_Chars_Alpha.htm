Control Characters: Alpha command (View menu)

 

This command displays control characters in the character area as their corresponding alphabetic character.  This command is not available if the character area is not displayed.  It is also not used when the IBM/OEM or the EBCDIC character set is displayed.

 

For example, a byte with value zero is displayed as '@', value 1 as 'A', etc.  They are usually differentiated by from normal ASCII characters by being displayed in red, although this depends on the colour scheme in use (see Colour Options page).

 

Alternative ways to display control characters are provided by the Control Chars None, and Control Chars C/C++ commands.

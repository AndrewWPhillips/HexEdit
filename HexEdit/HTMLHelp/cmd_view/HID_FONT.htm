Font command (View menu)

 

Use this command to change the font that is used to display the hex and character areas in the active window. This command brings up the standard font Choose Font dialog to allow you to select any available font in any size and style.

 

To quickly change the font use the Font_tool and the Font Size tool.  To quickly increase or decrease the font size use the Increase Font Size command or the Decrease Font Size command.

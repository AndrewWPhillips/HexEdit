Character Set Code Page command (View menu)

 

This displays the character using the current code page.  The code page will be the last code page used or the default code page if none was previously used.

 

To select the code page to use the Code Page drop-down list Document Display Options page.  You can also quickly select to display common code pages using commands for common code pages on the View/Code Page sub-menu.

 

This command is only available if the character area is displayed (see Show Char Area or Show Both Areas).

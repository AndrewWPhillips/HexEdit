Character Set ASCII+IBM/OEM command (View menu)

 

This displays characters (in the char area) using an IBM/OEM font. This displays graphics characters that were often used in MSDOS programs such as games.  This command is not available if the character area is not displayed (see Show Char Area or Show Both Areas).

 

Since the IBM PC hardware defined characters for all 256 different byte values all characters are displayed with some symbol (ie none are displayed just as a dot).  Furthermore, ASCII control characters are displayed as a graphics character so the options for control character display are not available.

 

When this option is on the current IBM/OEM font is used. This changes the set of fonts available when the Font command is used to change the current font.  Also with this command string searches (see Find Tool) and how text is copied to the clipboard (see Copy command) are performed using ASCII.

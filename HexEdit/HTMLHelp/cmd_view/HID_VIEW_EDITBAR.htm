Toolbars: Edit Bar command (View menu)

 

Use this command to toggle on or off the display of the Edit Bar. The Standard Toolbar (see View Standard Toolbar) and the Edit Bar are default toolbars which have buttons for many common commands in HexEdit. A check mark appears next to the menu item when the Edit Bar is displayed.

 

See Edit Bar for help on using the Edit Bar.  The Customize Commands page allows you to change the buttons on the Edit Bar.

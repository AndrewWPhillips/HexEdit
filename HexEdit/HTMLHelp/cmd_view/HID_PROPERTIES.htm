Properties command (View menu)

 

This displays the Properties dialog, which shows file information and values for the byte(s) at the cursor position in the currently active window. The dialog remains active even while editing (it is “modeless” in Windows jargon) – this allows you to easily see the properties as you move the cursor within a file or when you change to another window.

 

This dialog has the following pages:

Summary Page shows file size and time spent editing/viewing + edit/view comments, categories, keywords

File Properties the name, type, location, times etc

Character Properties value in octal, binary, etc and ASCII, Unicode or any code page representation

Integer Values value of word, double word etc, signed or unsigned, little- or big-endian

Floating-Point Values IEEE/IBM floating-point or C# Decimal values, 32 bit or 64 bit, little- or big-endian

Date/Time Values date/time values, including several time_t and Windows formats

 

Shortcuts

Mouse: Double click the “Hex, Decimal, Octal, Binary, Character” indicator of the status bar

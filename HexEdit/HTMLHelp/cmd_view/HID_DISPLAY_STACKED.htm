Display Stacked command (View menu)

 

This is a special display/edit mode where each byte is displayed as the characters stacked vertically.  The top character is the character for the byte in the current character set with the two hex digits representing the two nybbles of the byte beneath it.  To change the character set see Char Set ASCII, Char Set ANSI, Char Set IBM/OEM, and Char Set EBCDIC).  When the ASCII or ANSI character sets are displayed you can also change how control characters are displayed (see Control Chars None, Control Chars Alpha, and Control Chars C/C++).

Step 1. Customizing Toolbars

 

         

 

HexEdit like many programs provides several toolbars with lots of buttons which, to be honest, you will likely never use.  This may be because they invoke some esoteric mode (like EBCDIC character set display) or there are keyboard shortcuts that are faster (as for Cut, Copy and Paste).  However, there are undoubtedly many that you do (or should) use.  Why not put all these together on one toolbar?  Once you create the toolbar you can hide all the other toolbars to free up screen space.

 

Plus many capable commands languish in some obscure sub-menu or keyboard shortcut, never to see the light of day.  You can give them the opportunity to perform on the big stage too.

 

In this step of the tutorial we will create and customize a new toolbar.

 

You could modify an existing toolbar instead of creating a new one.  I don't recommend this because your changes could be lost if you upgrade to a new version of HexEdit (although generally toolbar customizations are preserved when upgrading).  At some stage you may also need to reset the toolbars which means changes to built-in toolbars are lost.

 

A way to save even more screen space is to drag all the toolbar buttons to the menu bar, instead of a new toolbar.  This assumes your screen is wide enough that you can fit all the buttons you want on the one line next to the main menus.  Also your customizations to the menu bar are lost when you install a new version of HexEdit or if you need to reset it.

 

 

Creating a Custom Toolbar

 

1. Open the Customize dialog (using the Tools/Customize menu item) and select the Toolbars (2nd) tab.

 

You can also open the Customize dialog by right-clicking any toolbar and selecting the "Customize" item at the bottom of the context (popup) menu.

 

When the Customize dialog is open all the main menus and toolbars are visible in the main window but do not do anything.  This allows you to reconfigure them without accidentally making changes to any file you have open..

 

2. Click the "New..." button and give the new toolbar a name such as "Mine" and click OK.

 

 

3. Find the new toolbar on the screen.  It may be hard to see as it doesn't have any buttons on it yet.

 

4. Display all the other toolbars (so you can copy from them) by selecting their checkbox in the list of Toolbars.

 

5. Hold down the Ctrl key and drag some buttons from the other toolbars to your new toolbar.

 

If you do not hold down Ctrl while dragging then the button is moved not copied.  That is, the button will disappear from the place you are dragging it.

 

A "+" appears next to the mouse pointer when you hold down the ctrl key to indicate that you are copying not moving.

 

Buttons you might consider are: Auto Fit, Hex/Decimal Address Toggle, Highlighter, Track Changes, plus the Show Area, Mark and Bookmarks drop downs.  The Color Scheme Tool, Hex and Decimal Jump Tools, and the Bookmarks tool are also very useful.

 

Press the Esc key while dragging if you change your mind and do not want to change anything.

 

 

The toolbar you have created is currently floating, which means it always appears on top of the main HexEdit window.  We will see below how to dock the toolbar so it is incorporated into the border of the main HexEdit window.

 

Floating toolbars can be arranged into single or multiple rows of buttons.  To adjust the shape (and hence the number of rows of buttons) of the toolbar click and drag the right border.

 

6. Now remove a button from your toolbar by dragging it anywhere except over a toolbar.

 

An "x" appears next to the mouse cursor to show that you are deleting.

 

If at any time you find you cannot modify a toolbar (ie the mouse cursor always shows an "x") then you need to click the "Reset All" button.  This will fix the problem but will also reset all the built-in toolbars to their original configuration.

 

Not only can you customize toolbars but you can customize the main menus (and popup menus as discussed in Step 3).  You can even drag menu items and whole sub-menus to toolbars and vice versa.  When you drag a menu item that has no image to a toolbar it is displayed as text - however, you can create an image for it as discussed below.

 

 

Button Types

 

Some of the toolbar buttons are menu buttons that display a menu when selected (eg the Show Area button on the Standard Toolbar).  These have a small down arrow next to the button image.  You can drag these buttons as normal to another toolbar and the menu goes with the button.  You can also click the button to open the menu and drag one of the commands within it.  Note that these menu buttons behave as "sub-menus" when dragged onto a menu.

 

Some toolbar buttons are command+menu buttons.  These execute a default command when clicked on but also have a menu available by clicking the separate down arrow to the right.  Examples are the Mark and Bookmark buttons on the Navigation Bar.  Again you can move the whole button or just individual buttons in the menu.

 

Some toolbars also have "tools" which are a combination of a text box with a drop down list.  An example is the Find tool which allows you to enter something to search for in the text box but also allows you to retrieve previous searches by selecting them from the drop-down list.  Another example is the bookmarks tool which displays the closest bookmark in the file and allows you to quickly jump to any bookmark in the file by selecting it from the list.  Since these tools contain text that can vary in length it is often useful to be able to change the tool's width.  How to do this is explained in the next section.

 

 

Resizing Tools

 

Another way to add to toolbars and menus is to drag buttons and tools from the Customize dialog itself, rather than copying from existing toolbars and menus.

 

7. Select the Commands (1st) tab in the Customize dialog.

 

 

8. In the Categories list scroll down to  "Find commands" near the bottom.

 

9. In the Commands list drag the Search tool (at the top of the list) to your new toolbar.

 

You do not need to hold down the Ctrl key when dragging from the Customize dialog Commands tab.

 

10. In the copy of the Find tool in your toolbar click the down-arrow (even though it appears disabled) at the right end to select the tool.

 

11. Move the mouse over the right hand border of the tool - the mouse cursor changes to a resize cursor.

 

12. Click and drag the right end to resize the Find tool to the width you desire.

 

Other tools like the hex jump tool can be resized in the same way.

 

 

Tidy Up

 

If you are finished customizing your toolbar it is probably time to tidy things up.

 

12. In the Toolbars tab deselect all toolbars except your new toolbar and the Menu Bar.

 

Note that however hard you click you can't turn off the Menu Bar.

 

13. Drag your toolbar (click and drag it by the title bar) underneath the menu bar to dock it there.

 

The toolbar shape and outline changes to indicate you have dragged it to a position to be docked.  Hold down the Ctrl key if you want the toolbar to remain floating, ie to prevent docking.

 

Docked toolbars can be repositioned and undocked by dragging the gripper (the double dotted line) at their left end.  You can also easily switch between docked and undocked by double-clicking the toolbar -- double-click on the gripper when docked or on the title-bar when undocked.

 

You can also dock toolbars vertically -- on the left or right side of the window.  However, when docked like this the tools revert to a simple button -- for example the Bookmarks tool becomes a button that invokes the Bookmarks dialog.

 

 

Creating New Buttons

 

For specific tasks you may want to add a command from a menu to a toolbar for quick and easy access.  Unfortunately, not all menu items have images.  You can display them as text in the toolbar but to save space you might want to create your own image.

 

Now we will add a command from the Template menu, the "Allow Editing" command, to the new toolbar.  (This is a command that I use a lot when working on a template.)

 

14. With the Customize dialog open click on the Template menu to show its commands.

 

If you currently don't have any files open you will see the "default" menus that are shown when HexEdit is first opened.  To see the full menus switch to the Menu tab in the Customize dialog and select "HexEd" in the drop down list entitled "Show Menus for".

 

15. Ctrl+drag (drag with the mouse while holding down the Ctrl key) the "Allow Editing" command near the bottom of the Template menu to your toolbar.

 

 

16. Right-click the "Allow Editing" button you just added to your toolbar and select "Image" from the context menu.

 

As this button does not have an image (yet) HexEdit opens the "Button Appearance" dialog.  Among other things this allows you to change the image for a toolbar button.  You can select from one of the predefined images, or click "New..." to create a new image in the image editor, or select a predefined image and click "Edit..." to modify it.

 

The image editor provides basic image editing abilities.  For more advanced editing use your favorite bitmap editor and use the clipboard to copy and paste the image into the button image editor.

 

17. Make sure "Image only" is selected and pick an image (eg the smiley face) and click OK.

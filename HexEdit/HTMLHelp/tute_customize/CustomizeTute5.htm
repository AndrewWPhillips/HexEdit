Step 5. Options Dialog

 

         

 

Sometimes the distinction between an "option" and a "customization" can be subtle.  In my opinion "customization" implies more intrinsic behavior that you might not normally expect to be able to change.  In this regard, I think the following facilities in the Options dialog are suitable for a customizations tutorial.

 

Configuring Color Schemes

 

On the surface color schemes look nice but underneath they offer a lot more utility than is immediately apparent.  We will create a color scheme for a particular file format in order to demonstrate some of the facilities.  This color scheme is only intended to be used with files of the (fictitious) "HEG" format.  By naming the color scheme "_HEG" (ie an underscore followed by the file extension) any file you open in HexEdit with that extension will automatically use that color scheme.

 

HexEdit may not always use the color scheme based on the file extension.  If you have opened a file in HexEdit before then HexEdit remembers the color scheme in use at that time.

 

1. Open the Options dialog (Tools/Options) and go to the "Colors" page.

2. Click the new scheme button on the left side (tip = "New Scheme).

3. In the New Scheme dialog enter a name of "_HEG" so it is associated with .HEG files.

4. In the "Copy Settings from" list select "ASCII Default" and click OK.

 

 

You have now created a new color scheme, but it is exactly the same as the "ASCII Default" scheme.  Now you need to modify the scheme using the controls on the right side of the dialog.  By default the _HEG scheme is now selected in the list of schemes on the left, so the changes below will be applied to that scheme.

 

First we need to tidy up the colors used for the color groups.  These are the last three entries in the list on the right with names (assuming you have not modified the ASCII Default scheme) of "ASCII Text", "Special (TAB,LF,CR)" and "Other".  We don't need the 2nd of these since these characters have no significance to this file format.  We also do not need bright red for "Other" as .HEG is a binary format and can validly contain any byte value so we don't want any particular value to stand out.

 

5. Select the "Special (TAB,LF,CR)" entry near the end of the list and click the delete button (2nd button from the top on the right).

6. Select the "Other" entry at the end of the list then select dark blue swatch from the "Color" control.

 

We now have two color groups: one that displays ASCII text in black and one that displays everything else in dark blue.  We will now add a new color group that shows bytes with values 1, 2, or 10 in purple.

 

7. Click in the "Name" edit box at the top and type "Test".

8. Change the "Color" control to purple by selecting from the color swatches.

9. In the "Values" edit box enter "1, 2, 10".

10. Click the new color group button at the top right to add the new group.

11. Click the up arrow button to move the new color group above the "Other" group.

 

 

Color groups are scanned from top to bottom to find a match so if you don't move the new group above the "Other" group it will have no effect.  The "Other" group has a byte range of 0 to 255 (ie all possible values) and will match everything.

 

Finally, as a visual clue about which color scheme is in use I also like to adjust the background color, keeping it very light, of course.

 

12. Scroll to the top of the list and select the "Background" entry.

13. Open the "Color" control and select the "Other" button at the bottom to bring up the Colors dialog.

14. In the Colors dialog select the "Custom" page.

15. Enter 240, 255, and 240 into the red, green and blue fields.  (This RGB value is for a light green.)

 

 

16. Click OK to close the Colors dialog. Click OK to close the Options dialog saving your changes.

 

Now we need to open a HEG file to make sure the color scheme works.  There should be a file called "tutorial.HEG" in the location that you installed HexEdit - typically "C:\Program Files\HexEdit".

 

17. Open tutorial.HEG from the directory where you installed HexEdit.

 

If you have already had this file open in HexEdit (perhaps after running the Advanced Template Tutorial) then HexEdit will remember the color scheme in use when you last used it, and will not use the _HEG color scheme.

 

To see the effect of the color scheme you can try flipping between different schemes.

 

18. From the Color Scheme tool select "ASCII Default" from the drop-down list.

 

The Color Scheme tool is on the Standard toolbar.  You may also have added it to you custom toolbar in Step 1 of this tutorial.

 

 

19. Change back to the _HEG color scheme in the same way.

 

 

Using File Type Filters

 

Most software deals with only a handful of different types of files or at most a few dozen.  Hence you are normally presented with a small, fixed number of file types in the "Files of type:" drop down list.

 

The "Files of type:" drop down list appears near the bottom of the standard file open dialog - just above the "Open as read-only" checkbox in HexEdit.

 

 

A hex editor is different from other software in that it is explicitly designed to be able to view and modify any type of file.  Nevertheless when using HexEdit you will usually only deal with a limited set of file types, perhaps occasionally needing to deal with a new type.

 

This is why HexEdit's File Type Filter list is very useful.  You have exact control over what appears in the "Files of Type" list that is shown in all Windows file open and save dialogs.  Further, you can easily turn on and off filters in the list as needs dictate.

 

To demonstrate we will add a filter for our (fictitious) graphics file format.  Files of this type have the extension "HEG".

 

21. Open the options dialog (Tools/Options...) and select the Filters tab.

22. Click the "new" button (tip = "New Filter") to the left of the "delete" button.

 

A new filter is added at the current position in the list and is enabled and selected ready for you to enter a description.

 

23. In the Filter Name column enter "HexEdit Graphics format (.heg)".

24. Press the Tab key to move to the "Filter" column and enter "*.heg".

25. Click the down arrow (right-most of the 4 buttons at the top) to move the filter below the "All Files" filter.

 

 

26. Click OK to close the Options dialog.

27. Open the Open dialog (File/Open...) to test the filter.

28. Select "HexEdit Graphics format" from the "Files of Type:" drop-down list.

 

You will now probably not see any files at all in the File Open dialog.

 

29. Browse to the directory where HexEdit is installed and you should see a file called "tutorial.heg".

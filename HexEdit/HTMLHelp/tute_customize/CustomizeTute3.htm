Step 3. Menu Customization

 

         

 

We saw in step 1 of this tutorial that configuring the main menu was closely tied to and similar to configuring the toolbars.  However, there are a few tricks to menus that you may need to know to change them.  First there are two types of configurable menus: Applications Frame (main) menus and Context (popup) menus.  (You may also see various context menus used throughout the program that are not configurable.)

 

Main Menus

 

An Application Frame menu contains the set of main menus that you see at the top of the HexEdit window.  There are just two of these:

 

A. Default menu contains the minimal set of menus you see when you first run HexEdit or when no files are open.

B. HexEd menu contains the full set of menus you see when one or more files are open.

 

1. Open the Customize dialog and select the Menu tab.

2. In the "Show Menus for" list try both the Default menu and HexEd options to see the two main menus.

You can customize main menus just like toolbars by dragging menu items around, and even whole sub-menus.

Popup Menus

 

A popup (or context) menu "pops up" when you right-click certain areas of the display.  These menus contain commands and can be customized just like toolbars and main menus.  There are currently 8 popup menus that are configurable:

 

Popup Menu

Accessed by right-clicking:

Status Bar

any part of the status bar (bottom of main window)

Window Tabs

the tab for a particular file in the tabs at the top (or bottom) of the main window

Address Area

the address area of the active file (to the left of the data area)

Hex Area

the hex area of the active file (or any hex digits in stacked mode)

Character Area

the character area (or any character in the top row in stacked mode)

Selection

some selected bytes (in either of the hex or character area)

Bookmarks

a bookmark in the active file (in either of the hex or character area)

Highlights

a part of the active file that is highlighted ( (in either of the hex or character area))

 

To demonstrate customization we will add the "Select All" command to the Window Tabs menu.

 

3. Open the Customize dialog and select the Menu tab.

4. In the "Select Context menu" list select the "Window Tabs" entry at the end of the list.

 

 

5. Find the window displaying the Window Tabs context menu, and move it next to the dialog.

6. Select the Commands tab in the Customize dialog.

7. Select "Edit" in the Categories list, then find "Select All" in the Commands list.

8. Drag "Select All" to the top of the "Window Tabs" menu.

 

In menus and toolbars it is often convenient to group commands together by placing "separators" between the groups.  When customizing menus and toolbars you can easily add separators by using the "Start Group" command on the item below or to the right of the desired separator location.  To do this just right-click the button or menu item and select "Start Group" from the popup menu.

 

9. Right-click "Close" in the menu (the item below "Select All") and select "Start Group".

 

 

10. Close the Customize dialog and test the new menu item.

 

 

Menu Options

 

The Menu page also has options for how menus are drawn.  There are various options for how they are animated when they appear.  There is also an option to draw menus with shadows.  In my opinion the animations are not that useful but the menu shadows look nice.

 

The Options page of the Customize dialog also has options for "personalised menus".  When this option is on only menu items that you have used recently (as well as a few "important" menu items) are initially shown in a menu.  The full menus are shown when the chevron is clicked or (optionally) after a delay.  Many people find this behaviour annoying so it is off by default.

Open Template

 

This command opens an existing template.  All existing templates are available in the Template menu under the "Open by File Type" and "Open Other" sub-menus.  These list the templates associated with file extensions as well as any other templates.

 

Template files can be associated with a file extension by giving them a name preceded by an underscore.  For example, the template file "_BMP.XML" is the template for .BMP files.  When you open a file in HexEdit it will use a template for that file type if there is one (and if you haven't associated a different template with the file).

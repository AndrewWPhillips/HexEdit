Auto-Synchronise Tree and Data Toggle

 

Keeps the selection automatically synchronised in the tree (template) and data (hex) windows.  See Synchronise Tree and Data for more information.

 

Turning on auto-synchronisation is particularly useful when the template is displayed in a split window, since you can see the elements of the template and the data bytes simultaneously.  See Display Options page for information on using a split window for displaying the tree view.

Error: HexEdit has not been installed

 

The message "Hexedit has not been installed on this machine" can have several causes.  For some reason the installation is corrupt or incomplete.  This could be due to setup failing to complete successfully, or some of the files or registry entries being corrupt  or missing. This can occur if you just copy the HexEdit.exe file to a computer without running setup.exe.  If this happens with a new installation then it may be that an earlier version of HexEdit was installed at some time but not properly removed.

 

This problem is usually fixed by uninstalling and reinstalling HexEdit.  It's possible, that HexEdit cannot be uninstalled properly due to the uninstall information being lost or corrupted.  If this is the case then you need to uninstall manually by following this procedure:

 

1. Uninstall HexEdit if you can using the "Add/Remove Programs" option in the Control Panel

 

2. Remove all files from the directory where HexEdit.exe was installed, if any

 

You now need to delete all registry entries under HKEY_LOCAL_MACHINE\Software\ECSoftware using regedit using steps 3 to 8.  Be careful not to delete any other entries.

 

3. From the Start menu choose Run and type "regedit" into the text box and hit Enter.

 

4. Expand "HKEY_LOCAL_MACHINE" in the tree view on the left by clicking the plus (+) sign.

 

5. Expand "Software" underneath that.

 

6. Click on "ECSoftware" underneath that so that it is selected - ie. has blue background (in the default Windows color scheme).

 

7. Hit the Del key and click the "Yes" button to confirm deletion. (Make sure you only delete the "ECSoftware" entry.)

 

8. Close regedit.

 

9. Try installing HexEdit again

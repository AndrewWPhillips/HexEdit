Error: DLL not found

 

If you get a message like "This application has failed to start because MFC42.DLL was not found" when you run HexEdit it means that a DLL that HexEdit requires is missing.

 

If you get this message for MSVCP60.DLL then you need to reinstall HexEdit making sure that you select the option to install "Shared DLLs".  If you still have a problem (under NT/2000/XP) and you do not have rights to modify the Windows system directory you may need to reinstall while logged in as administrator or have your system administrator install HexEdit.

 

See Error: Procedure entry point ... not found for information about how to obtain MSVCRT.DLL and MFC42.DLL if you do not have them.

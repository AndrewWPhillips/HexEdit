I can't print page 34567

 

If a file would have more than 32768 pages when printed (ie it is of the order of 20 Mbytes or more in size) then (due to an overflow bug in MFC) you will not be able to print most or all of the pages using the "Pages" option in the "Print Range" box of the Print Dialog.  You also won't be able to see the pages in print preview.

 

If you want to print part of a large file then select the bytes that you want to print and use the "Selection" option in the "Print Range" box of the Print Dialog.

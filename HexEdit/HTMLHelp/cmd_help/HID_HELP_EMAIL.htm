Help via Email command (Help menu)

 

Use this command to send us email. This command is disabled if your system does not support MAPI (a system Windows uses to let programs send email). If your system supports MAPI but you never (again) connect to the Internet then there is not much point in using this command.

 

To be sure of a prompt response, address email to support@hexedit.com. For the latest information on bugs, enhancements and future directions visit the HexEdit web site at http://www.hexeditpro.com. You can quickly connect there from within HexEdit using the Help via the Web command.

 

Your email address is kept strictly confidential, and will never be made available or sold to anyone without your permission.

 

For more information see the Send Email to HexEdit Support dialog.

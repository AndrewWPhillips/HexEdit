Repair Customizations command (Help menu)

 

HexEdit has comprehensive customization facilities (see Customize command).  However the registry settings that store the customizations may become messed up if you have accidentally made changes such as deleting buttons or menu items, or HexEdit crashed while updating the registry, or you have upgraded from HexEdit 2.1.

 

This command allows you to return HexEdit customizations to the factory default settings.  You should do this if:

  • Tool bar buttons show the wrong image.

  • Menu items show the wrong image or description.

  • Standard tool bar buttons or menu items have disappeared.

  • Standard toolbars are missing.

  • Standard keystrokes like Ctrl+X no longer work.

  • You cannot assign a custom keystroke to a command.

Note that after using this command you will lose all customizations to toolbars, menus, popup (right-click) menus, mouse double-click events and keyboard.

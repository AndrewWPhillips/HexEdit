Save Macro dialog

 

 

This dialog allows you to save the current keystroke macro to a HexEdit macro (.HEM) file. This allows it to be reloaded and run at a later time (see the Load Macro button in the Macro Options Page). This dialog is invoked using the Save Current Macro button in the Macro Options Page.

 

Note that the macro is saved in the current HexEdit macro folder. To change where macros are saved click on the Change Macro Folder button in the Macro Options Page.

 

There are several special macro names.  If you save a macro with the name “AutoExec” then the macro will be run whenever HexEdit is started.  (For example, you can use this to always open a file when HexEdit is run.)  If a macro has a name starting with an underscore followed by a file extension then that macro is run whenever a file with that extension is opened.  For example a macro with the name "_EXE" is run whenever you open a .exe file.

 

Name (file name)

This field names the macro and is also used to generate the file name. Do not add “.HEM” to this, as it is done automatically.

Comments

Add comments that describe what the macro is for and how to use it here. These comments are displayed when the macro is loaded.

Halt Condition

When the macro is loaded this sets the default error level that will be used. This can be overridden after the macro is loaded unless the macro is run automatically (eg. the AutoExec macro) or as part of another (nested) macro.

Default number of replays

This sets the default number of replays for the macro when it is loaded. Like the Halt Condition setting this can be overridden unless the macro is run automatically.

Save

This button saves the current macro to a file with the specified name and options and then closes the dialog.

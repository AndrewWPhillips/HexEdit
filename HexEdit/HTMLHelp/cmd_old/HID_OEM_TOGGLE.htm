IBM/OEM Graphics Toggle (View menu)

 

This changes what sort of graphic characters (characters with values above 127) are displayed, when in ASCII mode. This is useful for displaying the graphic characters used in many MSDOS programs.  This is an old command that has been replaced by the Char Set IBM/OEM command.

 

This command is not available in EBCDIC mode (see EBCDIC Characters Toggle) since it only applies to extensions to ASCII. This command is also not available unless graphic characters are being displayed (see Graphic Characters Toggle).

 

When this option is off the current ANSI font is used, when on the current IBM/OEM font is used. This changes the set of fonts available when the Font command is used to change the current font.

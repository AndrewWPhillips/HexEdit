Graphic Characters Toggle (View menu)

 

This toggles the display of graphic characters, ie, characters not in the ASCII character set, ie, with values above 127.  This is an old command that has been replaced by the use of the character set commands (Char Set ASCII, Char Set ANSI, Char Set IBM/OEM). This command is not available in EBCDIC mode (see EBCDIC Characters Toggle).

 

When graphic characters are off then these are displayed as dots. When the graphic characters are on, the characters displayed for byte values above 127 depends on whether an ANSI or an OEM font is being used. This is controlled by IBM/OEM Graphics Toggle.

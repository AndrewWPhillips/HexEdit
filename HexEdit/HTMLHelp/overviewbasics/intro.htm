Introduction

 

HexEdit Pro is a binary file editor that looks similar to many other such programs. If you are already familiar with binary files and at least one Windows word-processor or editor then you can probably use most of HexEdit Pro’s features straight away, but you should at least scan the Quick Start.

 

If you are not familiar with running Windows software and/or this type of program you should read the following help topics. If you are unfamiliar with binary files then you may need to read the Overview of Binary Files section too.

 

You may prefer to learn how to use HexEdit Pro using our new video tutorials - see http://www.hexeditpro.com/videos.aspx.

 

If you use HexEdit Pro only occasionally you will find its extensive on-line help very useful. You can usually find help at any time by pressing F1. Pressing Shift+F1 enters help mode whence you may click on a button or other object you want help on. To get help on a menu item hold the mouse over it and press F1.  Dialogs also provide popup help for all their controls, which is accessed through the F1 key or the “?” button on their title bar, or by right-clicking.

 

Main Window

 

A clickable screen shot of the HexEdit Pro window is shown below.  Like most Windows applications it has a main window with a menu bar and toolbars just below the title bar and a status bar at the bottom of the window.  One or more binary files can be opened in the main window with a row of tabs at the top or bottom of the window showing the open files.  There are also several tool windows that you can open and drag anywhere on the screen or dock to the sides of the main window.

 

The Title Bar shows the normal buttons on the right (minimize, close, etc).  Clicking the icon on the left brings up the System Menu. The text in the title bar indicates the active file being edited.  There is an asterisk (*) next to the name if the file has been modified and not yet saved.

 

Below the title bar is the main menu and the toolbars.  Initially, with no file open, the default menu is displayed which has a minimal set of items -- to see all the menu items you need to open a file first.

 

By default only the Standard Toolbar and the Edit Bar are visible below the menu bar.  Right-click on any toolbar to select from a list of all the toolbars.  You can also select "Customize" to modify the menus or toolbars or even create new toolbars.  Note that you can drag a toolbar anywhere on the screen.  If you drag one near the edge of the main HexEdit Pro window then it will "dock" there.

 

Many toolbars include simple tools that allow quick access to powerful abilities.  The Hex Address tool and Decimal Address tool allow you to quickly enter and jump to an address - but they also provide many other facilities like a history of jumps, base (radix) conversion, expression evaluation, etc.  Also the Find Tool allows quick searches, the Color Scheme Tool allows you to quickly change the current color settings, etc.

 

There are also several tool windows (sometimes called modeless dialogs).  These windows, like toolbars, can float anywhere or be docked to the sides of the main window.  They include the Bookmarks window, Calculator, Properties Dialog, Find Dialog and HexEdit Pro Explorer.  When floating these windows will get out of the way of the cursor so you can see what you are doing (but like most features in HexEdit Pro this can be turned off).  You can show or hide any toolbar or tool window by right-clicking a toolbar and selecting from the popup menu.

 

 

At the bottom of the main window is the Status Bar which shows status information.  The left side is a message area which displays a short description of menu items and toolbar buttons as you move the mouse over them. The message area also sometimes displays a progress bar while a process is executing or shows a message indicating an unusual situation that is not serious enough to warrant displaying a message dialog.

 

To the right of the message area are several panes or indicators showing status information.  If you hover the mouse over a pane you will see a small tip window that explains its purpose.  Right-clicking a pane displays a context menu of related commands.  Also double-clicking many panes will toggle their mode.  For example, double-clicking the REC pane will start (or stop) macro recording.

 

When you have several files open then there is a row of tabs showing all the open windows.  By default this is shown at the top of the window below the toolbars but can also be configured to appear at the bottom (just above the status bar).  The text of the tab is the file name, but you can see the full path name in a tip window if you hold the mouse over the tab.  The tabs allow you to easily see what files you have open and switch between them.  Right-clicking a tab displays a popup menu of relevant commands.

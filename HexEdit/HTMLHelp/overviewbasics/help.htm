Help

 

HexEdit provides help at every turn, if you need it. As well as the standard help topics available from the Help menu there are status bar messages, tooltips, tips of the day, keyboard map, context-sensitive help and popup help for dialog controls.

 

The status bar displays a message when you move the mouse over a menu item or a toolbar button, which describes what it does. A short “tip” is also displayed if you hold the mouse over a toolbar button or an indicator on the status bar, which describes what it is for.

 

To get more information on a menu item, toolbar button, or any part of the HexEdit display you can press Shift+F1. This invokes a help mode where the context-sensitive help is shown for the next thing you click on.

 

To quickly get help on a toolbar button or menu item hold the mouse over it and press F1.

 

When you start HexEdit a new Tip of the Day is displayed. This can be disabled using the Show Tip of the Day check box in System General Options page. You can also see the latest tip using the Tip of the Day command on the Help menu. The tips are stored in a text file called HexEdit.tip, which is found in the same place as HexEdit.exe.

 

Dialog Help

 

Often the place you really need help is in some obscure dialog, which you rarely use. HexEdit provides extensive help in dialogs.

 

All HexEdit dialogs (except a few simple ones) have a button labelled “Help”. The simplest way to find out about a dialog is to click this button. The help topic displayed shows general information about the dialog as well as specific information about the controls in the dialog.

 

To get quick help on a particular control in a dialog click the “?” in the dialog’s title bar and click on the control. Alternatively, select the control with the mouse or arrows keys and press F1, or right-click the control and select What's This from the popup menu. A small description is displayed in a “tip” window.
